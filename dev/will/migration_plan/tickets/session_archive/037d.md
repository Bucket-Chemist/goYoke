# GOgent-037d: User Intent Capture Hook

## Description

Implement PostToolUse hook to capture user intents from AskUserQuestion tool responses. This creates symmetry in the learning system: **037 captures failures (sharp edges), 037d captures successes (user guidance)**.

## Design Intent

The GOgent system currently learns only from failures (sharp edges, routing violations). User intent capture enables learning from **positive signals** - explicit user decisions, preferences, and guidance that should inform future sessions.

## Intended End State

- PostToolUse hook triggers on AskUserQuestion completion
- User intents written to `.claude/memory/user-intents.jsonl`
- SessionID captured for session clustering
- `gogent-archive user-intents` displays captured data
- Zero latency impact on tool execution (<50ms)

## Dependencies

- GOgent-029a: UserIntent schema and QueryUserIntents() API
- GOgent-037: Sharp Edge Capture (establishes hook pattern)

### CLI Entry Point

```go
package main

import (
    "encoding/json"
    "fmt"
    "os"
    "time"

    "github.com/Bucket-Chemist/GOgent-Fortress/pkg/session"
)

func main() {
    if err := run(); err != nil {
        // Log error but don't fail the hook
        fmt.Fprintf(os.Stderr, "[gogent-capture-intent] %v\n", err)
        // Output empty JSON to not break hook chain
        fmt.Println("{}")
        os.Exit(0) // Don't fail - graceful degradation
    }
}

func run() error {
    // Parse hook input from STDIN
    var hookInput HookInput
    if err := json.NewDecoder(os.Stdin).Decode(&hookInput); err != nil {
        return fmt.Errorf("failed to parse hook input: %w", err)
    }

    // Extract intent from AskUserQuestion result
    intent, err := extractIntent(hookInput)
    if err != nil {
        return fmt.Errorf("failed to extract intent: %w", err)
    }

    // Write to JSONL
    if err := appendIntent(intent); err != nil {
        return fmt.Errorf("failed to write intent: %w", err)
    }

    // Output success (empty response, no hook modification needed)
    fmt.Println("{}")
    return nil
}
```

### Hook Input Structure

```go
type HookInput struct {
    SessionID string `json:"session_id"`
    Tool      struct {
        Name   string          `json:"name"`
        Input  json.RawMessage `json:"input"`
        Result json.RawMessage `json:"result"`
    } `json:"tool"`
}

type AskUserQuestionInput struct {
    Questions []struct {
        Question string `json:"question"`
        Header   string `json:"header,omitempty"`
        Options  []struct {
            Label       string `json:"label"`
            Description string `json:"description,omitempty"`
        } `json:"options,omitempty"`
        MultiSelect bool `json:"multiSelect,omitempty"`
    } `json:"questions"`
}

type AskUserQuestionResult struct {
    Answers map[string]string `json:"answers"`
}
```

### Intent Extraction

```go
func extractIntent(hookInput HookInput) (*session.UserIntent, error) {
    var input AskUserQuestionInput
    if err := json.Unmarshal(hookInput.Tool.Input, &input); err != nil {
        return nil, fmt.Errorf("failed to parse tool input: %w", err)
    }

    var result AskUserQuestionResult
    if err := json.Unmarshal(hookInput.Tool.Result, &result); err != nil {
        return nil, fmt.Errorf("failed to parse tool result: %w", err)
    }

    // Handle single question case (most common)
    if len(input.Questions) == 0 {
        return nil, fmt.Errorf("no questions in input")
    }

    q := input.Questions[0]
    response := ""
    for _, ans := range result.Answers {
        response = ans // Take first answer
        break
    }

    // Determine confidence based on response type
    confidence := "explicit"
    if len(q.Options) == 0 {
        confidence = "inferred" // Free-form response
    }

    return &session.UserIntent{
        Timestamp:  time.Now().Unix(),
        Question:   q.Question,
        Response:   response,
        Confidence: confidence,
        Context:    q.Header, // Use header as context
        Source:     "ask_user",
        SessionID:  hookInput.SessionID,
    }, nil
}
```

### Append to JSONL

```go
func appendIntent(intent *session.UserIntent) error {
    // Determine path
    projectDir := os.Getenv("GOGENT_PROJECT_DIR")
    if projectDir == "" {
        cwd, err := os.Getwd()
        if err != nil {
            return err
        }
        projectDir = cwd
    }

    intentsPath := filepath.Join(projectDir, ".claude", "memory", "user-intents.jsonl")

    // Ensure directory exists
    if err := os.MkdirAll(filepath.Dir(intentsPath), 0755); err != nil {
        return err
    }

    // Open file for append
    f, err := os.OpenFile(intentsPath, os.O_APPEND|os.O_CREATE|os.O_WRONLY, 0644)
    if err != nil {
        return err
    }
    defer f.Close()

    // Write JSONL line
    data, err := json.Marshal(intent)
    if err != nil {
        return err
    }

    _, err = f.WriteString(string(data) + "\n")
    return err
}
```

func TestExtractIntent_SingleQuestion(t *testing.T) {

## Why This Matters

This ticket creates **behavioral symmetry** in the learning system:

| Artifact | Captures | Source |
|----------|----------|--------|
| SharpEdge | What went wrong | 037 hook |
| UserIntent | What user wanted | 037d hook |

Without 037d, the system only learns from failures. With it, the system learns from both failures AND explicit user guidance - enabling preference tracking, workflow customization, and cross-session memory.

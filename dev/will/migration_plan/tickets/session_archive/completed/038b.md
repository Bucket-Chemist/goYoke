# GOgent-038b: Load and Index Sharp-Edges YAML

## Description
Load sharp-edges.yaml files from all agent directories and build searchable index for pattern matching.

## Implementation Intention
Enable remediation suggestions in blocking responses by matching current sharp edges against known patterns.

## Intended End State
- Function LoadSharpEdgesIndex() scans ~/.claude/agents/*/sharp-edges.yaml
- Parses YAML into SharpEdgeTemplate structs
- Builds searchable index by error type, file pattern, keywords
- Test coverage ≥80%

## Dependencies
- GOgent-035-R: Sharp edge CLI with hook response generation (provides blocking/warning infrastructure)

## Format Decision: YAML vs Go-Native Alternatives

**Decision: Keep YAML**

| Format | Library | Human Edit | Native Parse | Verdict |
|--------|---------|-----------|--------------|---------|
| YAML | `gopkg.in/yaml.v3` | Excellent | No | **Selected** |
| TOML | `github.com/pelletier/go-toml/v2` | Good | No | Viable alternative |
| JSON | `encoding/json` (stdlib) | Poor | Yes | Not suitable for human-edited config |

**Rationale:**
1. `sharp-edges.yaml` files are human-edited by agents/developers
2. YAML provides superior readability for multi-line solutions
3. `gopkg.in/yaml.v3` is mature, well-maintained, widely used
4. The external dependency cost is justified by usability

**If zero-external-deps required in future:** Convert to TOML with `go-toml/v2`

```go
// SharpEdgeTemplate represents a known sharp edge pattern from YAML
type SharpEdgeTemplate struct {
	ID          string   `yaml:"id"`
	ErrorType   string   `yaml:"error_type"`
	FilePattern string   `yaml:"file_pattern"`
	Keywords    []string `yaml:"keywords"`
	Description string   `yaml:"description"`
	Solution    string   `yaml:"solution"`
	Source      string   `yaml:"-"` // File path where loaded from
}

// SharpEdgeIndex provides fast lookup of templates
type SharpEdgeIndex struct {
	ByErrorType map[string][]SharpEdgeTemplate
	ByKeyword   map[string][]SharpEdgeTemplate
	All         []SharpEdgeTemplate
}
```

### Implementation Pattern
```go
func LoadSharpEdgesIndex(agentDirs []string) (*SharpEdgeIndex, error) {
	index := &SharpEdgeIndex{
		ByErrorType: make(map[string][]SharpEdgeTemplate),
		ByKeyword:   make(map[string][]SharpEdgeTemplate),
		All:         []SharpEdgeTemplate{},
	}

	for _, dir := range agentDirs {
		yamlPath := filepath.Join(dir, "sharp-edges.yaml")

		// Skip if doesn't exist
		if _, err := os.Stat(yamlPath); os.IsNotExist(err) {
			continue
		}

		// Read YAML
		data, err := os.ReadFile(yamlPath)
		if err != nil {
			// Log warning but continue
			fmt.Fprintf(os.Stderr, "Warning: Failed to read %s: %v\n", yamlPath, err)
			continue
		}

		// Parse YAML
		var templates []SharpEdgeTemplate
		if err := yaml.Unmarshal(data, &templates); err != nil {
			fmt.Fprintf(os.Stderr, "Warning: Failed to parse %s: %v\n", yamlPath, err)
			continue
		}

		// Index templates
		for _, tmpl := range templates {
			tmpl.Source = yamlPath
			index.All = append(index.All, tmpl)

			// Index by error type
			index.ByErrorType[tmpl.ErrorType] = append(
				index.ByErrorType[tmpl.ErrorType], tmpl)

			// Index by keywords
			for _, keyword := range tmpl.Keywords {
				index.ByKeyword[keyword] = append(
					index.ByKeyword[keyword], tmpl)
			}
		}
	}

	return index, nil
}
```

### Expected sharp-edges.yaml Format
```yaml
- id: "python-pro-001"
  error_type: "TypeError"
  file_pattern: "*.py"
  keywords: ["type assertion", "interface", "bool"]
  description: "Type assertion on already-typed field"
  solution: "Use direct field access instead of type assertion. Field is already bool, not interface{}."

- id: "go-pro-042"
  error_type: "nil_pointer"
  file_pattern: "pkg/routing/*.go"
  keywords: ["map access", "routing", "tier"]
  description: "Accessing routing tier map without checking if key exists"
  solution: "Use two-value form: tier, ok := tierMap[key]; if !ok { ... }"
```

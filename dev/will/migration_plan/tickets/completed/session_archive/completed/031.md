# GOgent-031: Implement File Archival

## Description
Archive transcript, pending learnings, and violations to timestamped files in session-archive/ directory.

## Implementation Intention
Preserve full session history for analysis while clearing pending files for next session.

## Intended End State
- ArchiveSession() copies transcript and moves learnings/violations
- Timestamped filenames (session-{timestamp}.jsonl)
- CleanupTempFiles() removes tool counters and temp logs
- Test coverage ≥80%

# GOgent-042: Behavioral Simulation CI/CD Pipeline

## Description

Extend the existing simulation testing infrastructure with **behavioral testing** that validates multi-turn session sequences, cross-session memory, and concurrent agent scenarios. This creates a foundation for end-to-end learning pipeline validation.

## Problem Statement

Current simulation testing validates:
- ✅ Individual hook behaviors (PreToolUse, PostToolUse, SessionEnd)
- ✅ Deterministic fixture scenarios
- ✅ Fuzz testing with invariant checking

What's **missing**:
- ❌ Multi-turn session simulation (tool call sequences)
- ❌ Cross-session context propagation (handoff → next session)
- ❌ Concurrent agent chaos testing (shared state correctness)
- ❌ System-level behavioral invariants

## Scope Decisions

### In Scope (This Ticket)
- Session replay harness for multi-turn sequences
- Behavioral invariants that are testable NOW (B1, B4, B5, B6, B7)
- Chaos testing with shared-key contention
- CI/CD pipeline for Levels 1-4

### Out of Scope (Future Ticket 042b)
- Week Simulation (requires GOgent-041b `AggregateWeeklyIntents`)
- B2 invariant: code_snippet (requires GOgent-037b)
- B3 invariant: Category field (requires GOgent-041)

**Rationale:** These features depend on pending tickets. Implementing stubs would create false confidence. Better to scope down and deliver working tests now.

### Type Definitions

**Note:** We create `ReplayEvent` as a NEW type, distinct from the existing `SessionEvent` (which represents SessionEnd hook input). These serve different purposes:

- `SessionEvent`: Input for SessionEnd hook (existing)
- `ReplayEvent`: Complete tool use cycle for replay testing (new)

```go
// ReplayEvent represents a complete tool use cycle for session replay.
// Distinct from SessionEvent (SessionEnd input) and ToolEvent (PreToolUse input).
type ReplayEvent struct {
    // Timing
    Timestamp int64 `json:"ts"`

    // Hook routing
    HookType string `json:"hook_type"` // "PreToolUse" or "PostToolUse"

    // Tool identification (consistent with ToolEvent)
    ToolName  string                 `json:"tool_name"`
    ToolInput map[string]interface{} `json:"tool_input"`

    // Response (PostToolUse only)
    ToolResponse map[string]interface{} `json:"tool_response,omitempty"`
    Success      bool                   `json:"success"`

    // Optional: expected hook output for this event
    ExpectedDecision string `json:"expected_decision,omitempty"` // "block", "allow", ""
}

// ReplaySession represents a complete agent session for replay testing.
type ReplaySession struct {
    ID           string              `json:"session_id"`
    Description  string              `json:"description"`
    Events       []ReplayEvent       `json:"events"`
    Expected     ReplayExpectations  `json:"expected"`
}

// ReplayExpectations defines what should result from replaying the session.
type ReplayExpectations struct {
    // Artifact counts
    SharpEdgesCreated  int  `json:"sharp_edges_created"`
    BlockingResponses  int  `json:"blocking_responses"`
    HandoffCreated     bool `json:"handoff_created"`

    // File checks: path -> list of substrings that must appear
    FileContains map[string][]string `json:"file_contains,omitempty"`

    // Files that must exist after replay
    FilesCreated []string `json:"files_created,omitempty"`
}
```

### Session Replayer with State Management

**Key insight from review:** The existing runner resets `.claude/` between scenarios. Session replay needs persistent state WITHIN a session but isolation BETWEEN sessions.

```go
package harness

import (

// SessionReplayer executes recorded sessions against the hook system.
type SessionReplayer struct {
    validatePath  string
    archivePath   string
    sharpEdgePath string
    fixturesDir   string
    tempDir       string // Base temp directory
}

// NewSessionReplayer creates a replayer with paths to CLI binaries.
func NewSessionReplayer(validatePath, archivePath, sharpEdgePath, fixturesDir string) *SessionReplayer {
    return &SessionReplayer{
        validatePath:  validatePath,
        archivePath:   archivePath,
        sharpEdgePath: sharpEdgePath,
        fixturesDir:   fixturesDir,
    }
}

// ReplayAll loads and replays all sessions from the fixtures directory.
func (r *SessionReplayer) ReplayAll() ([]ReplayResult, error) {
    sessions, err := r.loadSessions()
    if err != nil {
        return nil, fmt.Errorf("load sessions: %w", err)
    }

    var results []ReplayResult
    for _, session := range sessions {
        result := r.ReplaySession(session)
        results = append(results, result)
    }

    return results, nil
}

// ReplaySession executes a single session and validates expectations.
func (r *SessionReplayer) ReplaySession(session ReplaySession) ReplayResult {
    result := ReplayResult{
        SessionID: session.ID,
    }

    // KEY: Create fresh temp directory for THIS session
    // State accumulates within session, but each session starts clean
    sessionTempDir, err := os.MkdirTemp("", "replay-"+session.ID+"-")
    if err != nil {
        result.Error = err
        return result
    }
    defer os.RemoveAll(sessionTempDir) // Cleanup after session

    // Initialize required directories
    os.MkdirAll(filepath.Join(sessionTempDir, ".claude", "memory"), 0755)
    os.MkdirAll(filepath.Join(sessionTempDir, ".gogent"), 0755)

    // Track metrics across events
    var blockingCount int
    var sharpEdgesCreated int

    // Execute events IN ORDER - state accumulates
    for i, event := range session.Events {
        output, err := r.executeEvent(event, sessionTempDir, session.ID)
        if err != nil {
            result.Error = fmt.Errorf("event %d: %w", i, err)
            return result
        }

        // Parse output for metrics
        var hookResp map[string]interface{}
        if json.Unmarshal([]byte(output), &hookResp) == nil {
            if decision, _ := hookResp["decision"].(string); decision == "block" {
                blockingCount++
            }
        }

        // Check for sharp edge creation after PostToolUse
        if event.HookType == "PostToolUse" {
            pendingPath := filepath.Join(sessionTempDir, ".claude", "memory", "pending-learnings.jsonl")
            if content, err := os.ReadFile(pendingPath); err == nil {
                sharpEdgesCreated = countJSONLLines(content)
            }
        }

        // Validate per-event expectation if specified
        if event.ExpectedDecision != "" {
            actualDecision, _ := hookResp["decision"].(string)
            if actualDecision != event.ExpectedDecision {
                result.EventErrors = append(result.EventErrors,
                    fmt.Sprintf("event %d: expected decision %q, got %q",
                        i, event.ExpectedDecision, actualDecision))
            }
        }
    }

    // Validate session-level expectations
    result.SharpEdgesCreated = sharpEdgesCreated
    result.BlockingResponses = blockingCount
    result.Passed = r.validateExpectations(session.Expected, sessionTempDir, sharpEdgesCreated, blockingCount)

    return result
}

// executeEvent runs the appropriate CLI for a single event.
func (r *SessionReplayer) executeEvent(event ReplayEvent, tempDir, sessionID string) (string, error) {
    var cmdPath string
    var input interface{}

    switch event.HookType {
    case "PreToolUse":
        cmdPath = r.validatePath
        input = ToolEvent{
            ToolName:      event.ToolName,
            ToolInput:     event.ToolInput,
            SessionID:     sessionID,
            HookEventName: "PreToolUse",
            CapturedAt:    event.Timestamp,
        }
    case "PostToolUse":
        cmdPath = r.sharpEdgePath
        input = map[string]interface{}{
            "tool_name":       event.ToolName,
            "tool_input":      event.ToolInput,
            "tool_response":   event.ToolResponse,
            "session_id":      sessionID,
            "hook_event_name": "PostToolUse",
            "captured_at":     event.Timestamp,
        }
    default:
        return "", fmt.Errorf("unknown hook type: %s", event.HookType)
    }

    return r.runCLI(cmdPath, input, tempDir)
}
```

```go
// BehavioralInvariant represents a property that must hold across sessions.
type BehavioralInvariant struct {
    ID    string
    Name  string
    Check func(ctx *ReplayContext) (bool, string)
}

// ReplayContext provides access to session state for invariant checking.
type ReplayContext struct {
    TempDir           string
    SharpEdges        []map[string]interface{}
    FailureTrackerLog []map[string]interface{}
    Handoff           map[string]interface{}
    Config            struct {
        MaxFailures int
    }
}

// BehavioralInvariants defines properties that must hold.
// NOTE: B2 (code_snippet) and B3 (Category) deferred to 042b pending schema extensions.
var BehavioralInvariants = []BehavioralInvariant{
    {
        ID:   "B1",
        Name: "sharp_edges_have_required_fields",
        Check: func(ctx *ReplayContext) (bool, string) {
            for i, edge := range ctx.SharpEdges {
                errorType, _ := edge["error_type"].(string)
                if errorType == "" {
                    return false, fmt.Sprintf("sharp edge %d missing error_type", i)
                }

                failures, _ := edge["consecutive_failures"].(float64)
                if failures < 3 {
                    return false, fmt.Sprintf("sharp edge %d has consecutive_failures=%.0f, want >=3", i, failures)
                }

                timestamp, _ := edge["timestamp"].(float64)
                if timestamp == 0 {
                    // Also accept "ts" field name
                    timestamp, _ = edge["ts"].(float64)
                }
                if timestamp == 0 {
                    return false, fmt.Sprintf("sharp edge %d missing timestamp", i)
                }
            }
            return true, ""
        },
    },
    {
        ID:   "B4",
        Name: "handoff_preserves_schema_version",
        Check: func(ctx *ReplayContext) (bool, string) {
            if ctx.Handoff == nil {
                return true, "" // No handoff in this session
            }

            version, _ := ctx.Handoff["schema_version"].(string)
            if version != "1.1" {
                return false, fmt.Sprintf("schema_version=%q, want 1.1", version)
            }
            return true, ""
        },
    },
    {
        ID:   "B5",
        Name: "failure_tracker_counts_accurate",
        Check: func(ctx *ReplayContext) (bool, string) {
            // Count failures by composite key from tracker log
            keyCounts := make(map[string]int)
            for _, entry := range ctx.FailureTrackerLog {
                file, _ := entry["file"].(string)
                errorType, _ := entry["error_type"].(string)
                key := file + ":" + errorType
                keyCounts[key]++

            return true, ""
        },
    },
    {
        ID:   "B6",
        Name: "blocking_at_exact_threshold",
        Check: func(ctx *ReplayContext) (bool, string) {
            // Verify no blocking before threshold
            // This requires tracking blocking decisions during replay
            // (Validated during ReplaySession execution)

        ID:   "B7",
        Name: "jsonl_files_valid_after_writes",
        Check: func(ctx *ReplayContext) (bool, string) {
            // Check all JSONL files are valid (no corruption)
            jsonlFiles := []string{
                ".claude/memory/pending-learnings.jsonl",
                ".claude/memory/handoffs.jsonl",
                ".gogent/failure-tracker.jsonl",

            for _, relPath := range jsonlFiles {
                fullPath := filepath.Join(ctx.TempDir, relPath)

                if err != nil {

                lines := strings.Split(strings.TrimSpace(string(content)), "\n")

                    if line == "" {
                        continue
                    }
                    var obj interface{}
                    if err := json.Unmarshal([]byte(line), &obj); err != nil {
                        return false, fmt.Sprintf("%s line %d: invalid JSON: %v", relPath, i+1, err)
                    }
                }
            }
            return true, ""
        },
    },
```

### Chaos Runner with Shared-Key Testing

**Key fix from review:** Original design had each agent use unique keys. This tests isolated writes, not contention. Fixed by adding `SharedKeyRatio`.

```go
package harness

import (
    "fmt"
    "math/rand"
    "sync"
    "time"
)

// ChaosConfig controls chaos testing parameters.
type ChaosConfig struct {
    NumAgents        int           // Number of concurrent agents
    FailuresPerAgent int           // Failures each agent generates
    SharedKeyRatio   float64       // Ratio of agents sharing same key (0.0-1.0)
    MaxFailures      int           // Threshold for blocking (default: 3)
    Duration         time.Duration // Max test duration
    Seed             int64         // Random seed for reproducibility
}

// DefaultChaosConfig returns sensible defaults.
func DefaultChaosConfig() ChaosConfig {
    return ChaosConfig{
        NumAgents:        10,
        FailuresPerAgent: 20,
        SharedKeyRatio:   0.3,  // 30% of agents share keys (tests contention)
        MaxFailures:      3,
        Duration:         30 * time.Second,
        Seed:             time.Now().UnixNano(),
    }
}

// ChaosResult captures outcome from one agent.
type ChaosResult struct {
    AgentID       int
    File          string
    ErrorType     string
    FinalCount    int
    BlockedAt     int // 0 if never blocked
    WriteErrors   int
}

// ChaosReport summarizes chaos test outcome.
type ChaosReport struct {
    TotalAgents    int
    TotalEvents    int
    SharedKeyPairs int      // Number of agents that shared keys
    Passed         bool
    Errors         []string
    Duration       time.Duration
}

// ChaosRunner simulates concurrent agents with failures.
type ChaosRunner struct {
    config      ChaosConfig
    tempDir     string
    sharpEdge   string // Path to gogent-sharp-edge binary
    results     []ChaosResult
    resultsMu   sync.Mutex
}

// NewChaosRunner creates a chaos runner.
func NewChaosRunner(cfg ChaosConfig, tempDir, sharpEdgePath string) *ChaosRunner {
    return &ChaosRunner{
        config:    cfg,
        tempDir:   tempDir,
        sharpEdge: sharpEdgePath,
    }
}

// Run executes chaos testing.
func (c *ChaosRunner) Run() (*ChaosReport, error) {
    start := time.Now()
    rng := rand.New(rand.NewSource(c.config.Seed))

    // Generate agent assignments
    // Some agents get unique keys, some share keys (per SharedKeyRatio)
    assignments := c.generateAssignments(rng)

    // Setup temp directory
    if err := c.setupTempDir(); err != nil {
        return nil, fmt.Errorf("setup: %w", err)
    }

    // Run agents concurrently
    var wg sync.WaitGroup
    for i := 0; i < c.config.NumAgents; i++ {
        wg.Add(1)
        go func(agentID int, assignment KeyAssignment) {
            defer wg.Done()
            c.runAgent(agentID, assignment, rng.Int63())
        }(i, assignments[i])
    }

    // Wait with timeout
    done := make(chan struct{})
    go func() {
        wg.Wait()
        close(done)
    }()

    select {
    case <-done:
        // All agents finished
    case <-time.After(c.config.Duration):
        return nil, fmt.Errorf("chaos test timed out after %v", c.config.Duration)
    }

    return c.buildReport(start, assignments)
}

// KeyAssignment tracks what file/error an agent uses.
type KeyAssignment struct {
    File      string
    ErrorType string
    IsShared  bool // Whether this key is shared with other agents
}

// generateAssignments creates key assignments with SharedKeyRatio sharing.
func (c *ChaosRunner) generateAssignments(rng *rand.Rand) []KeyAssignment {
    assignments := make([]KeyAssignment, c.config.NumAgents)

    // Calculate how many agents share keys
    sharedCount := int(float64(c.config.NumAgents) * c.config.SharedKeyRatio)
    if sharedCount < 2 && c.config.SharedKeyRatio > 0 {
        sharedCount = 2 // Need at least 2 to share
    }

    // Shared keys: all use the same file/errorType
    sharedFile := "shared_contention.go"
    sharedError := "ContentionError"

    for i := 0; i < c.config.NumAgents; i++ {
        if i < sharedCount {
            // Shared key assignment
            assignments[i] = KeyAssignment{
                File:      sharedFile,
                ErrorType: sharedError,
                IsShared:  true,
            }
        } else {
            // Unique key assignment
            assignments[i] = KeyAssignment{
                File:      fmt.Sprintf("agent_%d_file.go", i),
                ErrorType: fmt.Sprintf("UniqueError%d", i),
                IsShared:  false,
            }
        }
    }

    // Shuffle to mix shared and unique agents
    rng.Shuffle(len(assignments), func(i, j int) {
        assignments[i], assignments[j] = assignments[j], assignments[i]
    })

    return assignments
}

// runAgent simulates one agent generating failures.
func (c *ChaosRunner) runAgent(agentID int, assignment KeyAssignment, seed int64) {
    rng := rand.New(rand.NewSource(seed))
    result := ChaosResult{
        AgentID:   agentID,
        File:      assignment.File,
        ErrorType: assignment.ErrorType,
    }

    for i := 0; i < c.config.FailuresPerAgent; i++ {
        // Simulate failure via sharp-edge CLI
        count, blocked, err := c.simulateFailure(assignment.File, assignment.ErrorType)
        if err != nil {
            result.WriteErrors++
            continue
        }

        result.FinalCount = count
        if blocked && result.BlockedAt == 0 {
            result.BlockedAt = i + 1
        }

        // Small random delay to increase interleaving
        // Range: 10-110 microseconds (documented, not magic)
        delay := 10 + rng.Intn(100)
        time.Sleep(time.Microsecond * time.Duration(delay))
    }

    c.resultsMu.Lock()
    c.results = append(c.results, result)
    c.resultsMu.Unlock()
}

// buildReport analyzes results for correctness.
func (c *ChaosRunner) buildReport(start time.Time, assignments []KeyAssignment) (*ChaosReport, error) {
    report := &ChaosReport{
        TotalAgents: c.config.NumAgents,
        Duration:    time.Since(start),
    }

    // Count shared key pairs
    for _, a := range assignments {
        if a.IsShared {
            report.SharedKeyPairs++
        }
    }

    // Count total events
    for _, r := range c.results {
        report.TotalEvents += c.config.FailuresPerAgent - r.WriteErrors
    }

    // Validate: Shared-key agents should see combined count
    sharedCounts := make(map[string]int)
    for _, r := range c.results {
        key := r.File + ":" + r.ErrorType
        if existing, ok := sharedCounts[key]; ok {
            // Multiple agents with same key - verify counts are consistent
            // Due to concurrency, counts may differ slightly but should be close
            if abs(r.FinalCount-existing) > c.config.FailuresPerAgent {
                report.Errors = append(report.Errors,
                    fmt.Sprintf("key %s: inconsistent counts %d vs %d", key, r.FinalCount, existing))
            }
        }
        sharedCounts[key] = r.FinalCount
    }

    // Validate: Check for JSONL corruption
    if err := c.validateJSONL(); err != nil {
        report.Errors = append(report.Errors, err.Error())
    }

    // Validate: Blocking should occur for shared keys
    sharedBlocked := false
    for _, r := range c.results {
        if r.BlockedAt > 0 {
            // Check if this is a shared key
            for _, a := range assignments {
                if a.File == r.File && a.ErrorType == r.ErrorType && a.IsShared {
                    sharedBlocked = true
                    break
                }
            }
        }
    }

    if report.SharedKeyPairs > 0 && !sharedBlocked {
        // With shared keys, blocking should eventually occur
        totalSharedFailures := report.SharedKeyPairs * c.config.FailuresPerAgent
        if totalSharedFailures >= c.config.MaxFailures {
            report.Errors = append(report.Errors,
                fmt.Sprintf("shared keys had %d total failures but no blocking occurred", totalSharedFailures))
```

### CI/CD Workflow

```yaml
# .github/workflows/simulation-behavioral.yml

name: Behavioral Simulation

on:
  push:
    branches: [master]
  pull_request:
    branches: [master]
  schedule:
    - cron: '0 4 * * 0'  # Weekly Sunday 4am UTC
  workflow_dispatch:
    inputs:
      chaos_agents:
        description: 'Number of agents for chaos testing'
        default: '10'
      chaos_shared_ratio:
        description: 'Ratio of agents sharing keys (0.0-1.0)'
        default: '0.3'

env:
  GO_VERSION: '1.25'

jobs:
  # ===========================================================================
  # Level 1: Unit Invariants (fast, every push)
  # ===========================================================================
  unit-invariants:
    name: Unit Invariants
    runs-on: ubuntu-latest
    timeout-minutes: 10

    steps:
      - uses: actions/checkout@v4
      - uses: actions/setup-go@v5
        with:
          go-version: ${{ env.GO_VERSION }}

      - name: Build CLIs
        run: make build-validate build-archive build-sharp-edge

      - name: Run Deterministic Tests
        run: make test-simulation-deterministic

      - name: Run Unit Tests
        run: go test -v -race ./test/simulation/harness/...

  # ===========================================================================
  # Level 2: Session Replay (fast, every push)
  # ===========================================================================
  session-replay:
    name: Session Replay
    runs-on: ubuntu-latest
    needs: unit-invariants
    timeout-minutes: 15

    steps:
      - uses: actions/checkout@v4
      - uses: actions/setup-go@v5
        with:
          go-version: ${{ env.GO_VERSION }}

      - name: Build CLIs
        run: make build-validate build-archive build-sharp-edge

      - name: Run Session Replay Tests
        run: make test-simulation-replay

      - name: Upload Replay Reports
        uses: actions/upload-artifact@v4
        if: always()
        with:
          name: replay-reports-${{ github.run_number }}
          path: test/simulation/reports/replay-*.json
          retention-days: 14

  # ===========================================================================
  # Level 3: Behavioral Properties (thorough, PRs only)
  # ===========================================================================
  behavioral-properties:
    name: Behavioral Properties
    runs-on: ubuntu-latest
    needs: session-replay
    if: github.event_name == 'pull_request'
    timeout-minutes: 20

    steps:
      - uses: actions/checkout@v4
      - uses: actions/setup-go@v5
        with:
          go-version: ${{ env.GO_VERSION }}

      - name: Build CLIs
        run: make build-validate build-archive build-sharp-edge

      - name: Run Behavioral Tests
        run: make test-simulation-behavioral

      - name: Upload Behavioral Reports
        uses: actions/upload-artifact@v4
        if: always()
        with:
          name: behavioral-reports-${{ github.run_number }}
          path: test/simulation/reports/behavioral-*.json
          retention-days: 14

  # ===========================================================================
  # Level 4: Chaos Testing (weekly or manual)
  # ===========================================================================
  chaos-testing:
    name: Chaos Testing
    runs-on: ubuntu-latest
    if: github.event_name == 'schedule' || github.event_name == 'workflow_dispatch'
    timeout-minutes: 30

    steps:
      - uses: actions/checkout@v4
      - uses: actions/setup-go@v5
        with:
          go-version: ${{ env.GO_VERSION }}

      - name: Build CLIs
        run: make build-validate build-archive build-sharp-edge

      - name: Run Chaos Tests
        env:
          CHAOS_AGENTS: ${{ github.event.inputs.chaos_agents || '10' }}
          CHAOS_SHARED_RATIO: ${{ github.event.inputs.chaos_shared_ratio || '0.3' }}
        run: make test-simulation-chaos

      - name: Upload Chaos Reports
        uses: actions/upload-artifact@v4
        if: always()
        with:
          name: chaos-reports-${{ github.run_number }}
          path: test/simulation/reports/chaos-*.json
          retention-days: 30

  # ===========================================================================
  # Aggregate Status
  # ===========================================================================
  status:
    name: Simulation Status
    runs-on: ubuntu-latest
    needs: [unit-invariants, session-replay]
    if: always()

    steps:
      - name: Check Required Jobs
        run: |
          if [ "${{ needs.unit-invariants.result }}" != "success" ]; then
            echo "::error::Unit invariants failed"
            exit 1
          fi
          if [ "${{ needs.session-replay.result }}" != "success" ]; then
            echo "::error::Session replay failed"
            exit 1
          fi
          echo "All required simulation tests passed!"
```

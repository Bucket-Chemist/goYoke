## 7. Hypotheses to Investigate

### H1: Missing Fullscreen/Alternate Buffer
Ink may not be using alternate screen buffer, causing output to accumulate rather than replace.

### H2: Dimension Read During Render
Reading `stdout.rows` during render may return stale/changing values mid-frame.

### H3: No Screen Clear on Resize
Resize events may trigger re-render without clearing previous frame artifacts.

### H4: React Key Instability
Viewport or message list may have unstable keys causing full re-mounts.

### H5: Height Calculation Cascade
Height changes propagate through component tree causing multiple partial renders.

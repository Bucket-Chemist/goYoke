## Deployment Steps

### 1. Install Binary

```bash
make install-archive
```

**Verify**:
```bash
which gogent-archive
# Expected: /home/username/.local/bin/gogent-archive
```

### 2. Test Binary Manually

```bash
echo '{"session_id":"deployment-test","timestamp":1234567890,"hook_event_name":"SessionEnd"}' | gogent-archive
```

**Expected**: JSON confirmation output

**If error**: Check stderr, verify `GOGENT_PROJECT_DIR` or run from project root.

### 3. Update Hook Configuration

**Locate hook config** (common paths):
- `~/.config/claude-code/hooks.toml`
- `~/.claude/config/hooks.json`
- Project `.claude/hooks.toml`

**Update**:
```toml
[hooks.SessionEnd]
# OLD:
# command = "~/.claude/hooks/session-archive.sh"

# NEW:
command = "gogent-archive"
```

**Save and close editor.**

### 4. Reload Claude Code Configuration

```bash
# Method varies by Claude Code version
# Option 1: Restart Claude Code
# Option 2: Send SIGHUP to reload config
pkill -HUP claude-code
```

### 5. Trigger Test Session

Start a new Claude Code session and perform any action to trigger SessionEnd:
- Type a message
- Exit session

### 6. Verify Handoff Generated

```bash
ls -lh .claude/memory/handoffs.jsonl
ls -lh .claude/memory/last-handoff.md
```

**Expected**: Both files exist with recent timestamps.

### 7. Verify Handoff Content

```bash
# Check JSONL format
cat .claude/memory/handoffs.jsonl | jq .

# Check markdown sections
grep "## Session Metrics" .claude/memory/last-handoff.md
```

**Expected**: Valid JSON, markdown has required sections.

### 8. Verify Artifact Archival

```bash
ls -lh .claude/memory/session-archive/
```

**Expected**: Timestamped learnings/violations files if artifacts existed.

### 9. Verify Metrics Match Bash

Compare last few sessions (if bash handoffs exist):
```bash
# Check tool call counts match
```

### 10. Monitor for Errors

Check logs:
```bash
tail -f ~/.gogent/hook-failures.log
```

**Expected**: No errors logged.

## Rollback Procedure

**If deployment fails or bugs discovered**:

### Step 1: Restore Bash Hook

```bash
# Restore hook config
# Edit hook config file, change:
command = "gogent-archive"
# Back to:
command = "~/.claude/hooks/session-archive.sh"
```

### Step 2: Reload Configuration

```bash
pkill -HUP claude-code
```

### Step 3: Test Bash Hook

```bash
echo '{"session_id":"rollback-test","timestamp":1234567890,"hook_event_name":"SessionEnd"}' | ~/.claude/hooks/session-archive.sh
```

**Expected**: Markdown handoff created.

### Step 4: Verify Rollback

Start new session, trigger SessionEnd, verify bash handoff created.

### Step 5: Report Issue

File bug report with:
- Go CLI error output
- Example SessionEnd JSON that failed
- Expected vs actual behavior
- Logs from `~/.gogent/hook-failures.log`

## Troubleshooting

### Issue: "command not found: gogent-archive"

**Cause**: Binary not in PATH

**Fix**:
```bash
# Check installation
which gogent-archive

# If not found, reinstall
make install-archive

# Verify PATH
echo $PATH | grep .local/bin

# If missing, add to PATH
export PATH="$HOME/.local/bin:$PATH"
```

### Issue: "Failed to parse SessionEnd event"

**Cause**: Invalid JSON on STDIN or timeout

**Fix**:
```bash
# Test with valid JSON
echo '{"session_id":"test","timestamp":123,"hook_event_name":"SessionEnd"}' | gogent-archive

# Check STDIN timeout (default 5s)
# Increase if needed (requires code change)
```

### Issue: "handoffs.jsonl not created"

**Cause**: Permission error or invalid project directory

**Fix**:
```bash
# Check permissions
ls -ld .claude/memory/

# Fix if needed
chmod 755 .claude/memory/

# Check project dir env var
echo $GOGENT_PROJECT_DIR

# Run from project root if unset
cd /path/to/project
gogent-archive < session.json
```

### Issue: "Metrics don't match bash"

**Cause**: Metrics parity test not run (GOgent-028b)

**Fix**:
```bash
# Run metrics parity test
go test ./test/integration -run TestMetricsParity

# If fails, fix pkg/session/metrics.go
```

### Issue: "Context not loaded in new session"

**Cause**: Markdown format incompatible with load-routing-context.sh

**Fix**:
```bash
# Run context loading compatibility test (GOgent-028f)
bash test/compatibility/context_loading_test.sh

# If fails, fix pkg/session/handoff_markdown.go
```

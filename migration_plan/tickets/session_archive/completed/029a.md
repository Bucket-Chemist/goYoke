# GOgent-029a: User Intent Capture Schema & Storage

## Description

Create the UserIntent learning artifact type to capture user decision inputs, responses, and expressed preferences during sessions. This addresses the critical gap: "Where am I capturing user decision inputs - how am I storing responses - user intent etc?"

## Design Intent

The existing learning system captures **passive** artifacts (errors that happened, violations detected), but lacks **active** artifacts (user decisions, explicit preferences, expressed intent). UserIntent fills this gap by capturing:

1. What question was asked (from AskUserQuestion tool)
2. What the user answered
3. What action was taken based on the answer
4. Confidence level (explicit response vs inferred)

This enables systems-architecture-reviewer to understand **user preferences** alongside **system learnings**.

## Intended End State

- UserIntent struct defined in pkg/session/handoff_artifacts.go
- user-intents.jsonl file for storage
- loadUserIntents() function following loadPendingLearnings() pattern
- QueryUserIntents() function following QuerySharpEdges() pattern
- UserIntentFilters struct for filtering
- HandoffConfig extended with UserIntentsPath
- HandoffArtifacts extended with UserIntents slice
- Test coverage ≥90%

## Dual-Write Architecture Pattern

**This ticket follows the established 028k-o dual-write pattern exactly:**

```
Session runtime (hooks):
  Capture event → Append to user-intents.jsonl
                  (separate file, accumulates during session)

SessionEnd (gogent-archive):
  LoadArtifacts() → loadUserIntents() → HandoffArtifacts.UserIntents
                  → Embedded in per-session handoff record
                  → Appends to handoffs.jsonl (source of truth)
                  → Renders last-handoff.md (human view)
```

**Why dual-write?**
- **Session integrity**: Each handoff record is self-contained (can `show <id>`)
- **Cross-session queries**: Separate file enables `user-intents --since 7d`
- **Weekly rotation**: 029f can archive user-intents.jsonl independently
- **Pattern consistency**: Same as pending-learnings.jsonl (SharpEdges)

**Key constraint**: Follow `loadPendingLearnings()` pattern exactly for `loadUserIntents()`.

```go
// loadUserIntents reads user intents from user-intents.jsonl
func loadUserIntents(path string) ([]UserIntent, error) {
    file, err := os.Open(path)
    if err != nil {
        if os.IsNotExist(err) {
            return []UserIntent{}, nil // Missing file is normal
        }
        return nil, fmt.Errorf("failed to open %s: %w", path, err)
    }
    defer file.Close()

    var intents []UserIntent
    scanner := bufio.NewScanner(file)

    for scanner.Scan() {
        line := strings.TrimSpace(scanner.Text())
        if line == "" {
            continue
        }

        var intent UserIntent
        if err := json.Unmarshal([]byte(line), &intent); err != nil {
            continue // Skip malformed lines
        }
        intents = append(intents, intent)
    }

    if err := scanner.Err(); err != nil {
        return nil, fmt.Errorf("error reading %s: %w", path, err)
    }

    return intents, nil
}
```

### Query Function

**File: pkg/session/query.go** (add after QuerySharpEdges)

```go
// UserIntentFilters defines filter criteria for user intents
type UserIntentFilters struct {
    Source     *string // Filter by capture source (ask_user, hook_prompt, manual)
    Confidence *string // Filter by confidence level (explicit, inferred, default)
    HasAction  bool    // Only return intents with ActionTaken != ""
    Since      *int64  // Filter by timestamp
    Limit      int     // Maximum results (0 = unlimited)
}

// QueryUserIntents retrieves user intents with optional filters
// Returns all intents if no filters specified
//
// Example:
//   q := NewQuery("/project/dir")
//   source := "ask_user"
//   intents, err := q.QueryUserIntents(UserIntentFilters{
//       Source:    &source,
//       HasAction: true,
//   })
func (q *Query) QueryUserIntents(filters UserIntentFilters) ([]UserIntent, error) {
    intentsPath := filepath.Join(q.ProjectDir, ".claude", "memory", "user-intents.jsonl")

    file, err := os.Open(intentsPath)
    if err != nil {
        if os.IsNotExist(err) {
            return []UserIntent{}, nil
        }
        return nil, fmt.Errorf("failed to open user intents: %w", err)
    }
    defer file.Close()

    var intents []UserIntent
    scanner := bufio.NewScanner(file)

    for scanner.Scan() {
        line := strings.TrimSpace(scanner.Text())
        if line == "" {
            continue
        }

        var intent UserIntent
        if err := json.Unmarshal([]byte(line), &intent); err != nil {
            continue
        }

        // Apply filters
        if filters.Source != nil && intent.Source != *filters.Source {
            continue
        }
        if filters.Confidence != nil && intent.Confidence != *filters.Confidence {
            continue
        }
        if filters.HasAction && intent.ActionTaken == "" {
            continue
        }
        if filters.Since != nil && intent.Timestamp < *filters.Since {
            continue
        }

        intents = append(intents, intent)

        if filters.Limit > 0 && len(intents) >= filters.Limit {
            break
        }
    }

    if err := scanner.Err(); err != nil {
        return nil, fmt.Errorf("error reading user intents: %w", err)
    }

    return intents, nil
}
```

```go
// FormatUserIntent formats a single user intent as a markdown line
func FormatUserIntent(intent UserIntent) string {
    // Confidence badge
    badge := map[string]string{
        "explicit": "💬",
        "inferred": "🤔",
        "default":  "⚪",
    }[intent.Confidence]
    if badge == "" {
        badge = "❓"
    }

    // Base format
    formatted := fmt.Sprintf("%s **Q:** %s\n   **A:** %s",
        badge,
        truncateString(intent.Question, 80),
        truncateString(intent.Response, 100),
    )

    // Add action if present
    if intent.ActionTaken != "" {
        formatted += fmt.Sprintf("\n   ➡️ Action: %s", truncateString(intent.ActionTaken, 80))
    }

    return formatted
}

// FormatUserIntents formats multiple user intents as a markdown section
func FormatUserIntents(intents []UserIntent) string {
    if len(intents) == 0 {
        return ""
    }

    var sb strings.Builder
    sb.WriteString("## User Intents\n\n")

    for _, intent := range intents {
        sb.WriteString(FormatUserIntent(intent))
        sb.WriteString("\n\n")
    }

    return sb.String()
}

// truncateString truncates a string to maxLen with ellipsis
func truncateString(s string, maxLen int) string {
    if len(s) <= maxLen {
        return s
    }
    return s[:maxLen-3] + "..."
}
```

## Why This Matters

UserIntent is the **missing artifact type** that transforms GOgent from a passive error logger to an active learning companion:

1. **User preference memory**: Explicit choices (auth method, formatting preferences) persist across sessions
2. **Decision audit trail**: Know why certain choices were made ("User said use JWT")
3. **Agentic context**: systems-architecture-reviewer can query "What did the user prefer for X?"
4. **Confidence tracking**: Distinguish explicit answers from inferred/default behaviors
5. **Action correlation**: Link user decisions to implementation actions

Without UserIntent, the learning system captures only **what went wrong** but not **what the user wanted**. This ticket completes the picture.

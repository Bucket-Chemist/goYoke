# GOgent-028e: Installation & PATH Configuration

## Description
Create production-ready installation workflow with Makefile targets for global install/uninstall, PATH verification, and user-facing documentation.

## Implementation Intention
Make gogent-archive installation as simple as `make install`.

## Intended End State
- Single `make install` command installs all CLI tools
- `~/.local/bin` added to PATH if missing
- Uninstall target for cleanup
- User documentation updated

## Implementation Details

### Makefile Additions
Add to `Makefile`:

```makefile
.PHONY: install uninstall check-path

install: build-validate build-archive build-sharp-edge check-path
	@echo "Installing GOgent-Fortress CLIs to ~/.local/bin/..."
	mkdir -p ~/.local/bin
	cp bin/gogent-validate ~/.local/bin/gogent-validate
	cp bin/gogent-archive ~/.local/bin/gogent-archive
	cp bin/gogent-sharp-edge ~/.local/bin/gogent-sharp-edge
	chmod +x ~/.local/bin/gogent-*
	@echo "✅ Installed gogent-validate, gogent-archive, gogent-sharp-edge"
	@echo ""
	@$(MAKE) check-path

check-path:
	@if echo $$PATH | grep -q "/.local/bin"; then \
		echo "✅ ~/.local/bin is in PATH"; \
	else \
		echo "⚠️  ~/.local/bin is NOT in PATH"; \
		echo "Add this to your ~/.bashrc or ~/.zshrc:"; \
		echo "    export PATH=\"\$$HOME/.local/bin:\$$PATH\""; \
		echo "Then run: source ~/.bashrc"; \
	fi

uninstall:
	@echo "Uninstalling GOgent-Fortress CLIs from ~/.local/bin/..."
	rm -f ~/.local/bin/gogent-validate
	rm -f ~/.local/bin/gogent-archive
	rm -f ~/.local/bin/gogent-sharp-edge
	@echo "✅ Uninstalled all CLIs"
```

### CLAUDE.md Update
Add installation section to project `CLAUDE.md`:

```markdown
## Installation

### Build and Install

```bash
# Build all binaries
make build

# Install to ~/.local/bin (recommended)
make install

# Verify installation
which gogent-validate gogent-archive gogent-sharp-edge
```

### PATH Configuration

If `~/.local/bin` is not in your PATH:

```bash
# Add to ~/.bashrc or ~/.zshrc
export PATH="$HOME/.local/bin:$PATH"

# Reload shell config
source ~/.bashrc  # or source ~/.zshrc
```

### Uninstall

```bash
make uninstall
```

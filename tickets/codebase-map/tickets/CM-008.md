### Test Structure

**File**: `cmd/gogent-codebase-extract/extractor_test.go`

```go
func TestGoExtraction(t *testing.T) {
    tests := []struct {
        name     string
        fixture  string
        wantSyms []ExpectedSymbol
    }{
        {
            name:    "simple_function",
            fixture: "testdata/go/simple.go",
            wantSyms: []ExpectedSymbol{
                {Name: "Hello", Kind: "function", Exported: true, ParamCount: 0},
                {Name: "add", Kind: "function", Exported: false, ParamCount: 2},
                {Name: "Config", Kind: "type", Exported: true},
                {Name: "Handler", Kind: "interface", Exported: true},
            },
        },
        {
            name:    "methods_with_receivers",
            fixture: "testdata/go/methods.go",
            wantSyms: []ExpectedSymbol{
                {Name: "Do", Kind: "method", Receiver: ptrStr("*Server")},
                {Name: "String", Kind: "method", Receiver: ptrStr("Config")},
            },
        },
        {
            name:    "import_classification",
            fixture: "testdata/go/imports.go",
            wantImports: &ExpectedImports{
                Internal: []string{"internal/routing"},
                External: []string{"github.com/smacker/go-tree-sitter"},
                Stdlib:   []string{"fmt", "os", "encoding/json"},
            },
        },
    }

    extractor := &GoExtractor{}
    for _, tt := range tests {
        t.Run(tt.name, func(t *testing.T) {
            source, err := os.ReadFile(tt.fixture)
            require.NoError(t, err)

            tree, err := ParseFile(source, extractor)
            require.NoError(t, err)

            symbols, imports, err := extractor.ExtractSymbols(tree, source)
            require.NoError(t, err)

            if tt.wantSyms != nil {
                assertSymbols(t, symbols, tt.wantSyms)
            }
            if tt.wantImports != nil {
                assertImports(t, imports, *tt.wantImports)
            }
        })
    }
}
```

### Test Categories

**3. Call graph** (`callgraph_test.go`):

**4. Output compliance** (`output_test.go`):

**5. Integration** (`integration_test.go`):

**6. Parse error reporting** (`edge_cases_test.go`):
- Files with syntax errors produce `error_count > 0` and `parse_errors` entries

- Partial extraction still returns symbols from valid portions of broken files

**7. Three-stage pipeline integration** (`pipeline_test.go`):
- Fixture repo (5 files, 2 modules) runs through Stage 1 → Stage 2 → Stage 3
- Enriched JSON: every symbol has `description`, `module_identity`, `complexity`, `is_entrypoint`, `architectural_role`

- Mermaid diagrams: each `.mmd` file starts with valid directive (`graph`, `classDiagram`, `flowchart`)

### Edge Case Tests

| Test | Input | Expected Behavior |
|------|-------|------------------|
| Empty file | `testdata/go/empty.go` | No symbols, no error |
| Syntax error | `testdata/go/broken.go` | Partial extraction (tree-sitter is error-tolerant) |
| Very large function | 500-line function | Extracts with correct line_start/line_end |
| Unicode identifiers | `func café() {}` | Extracts correctly |
| No module boundary | Single `.py` file (no `__init__.py`) | Treated as file-level module |
| Generated code | File with `// Code generated` | Skipped (unless --include-generated) |
| Binary file | `.png` in directory | Ignored by discovery |
| Symlink | Symlinked directory | Followed (matches default `filepath.Walk` behavior) |
| Parse errors | File with invalid syntax | `error_count > 0`, `parse_errors` populated, partial symbols extracted |
| Clean parse | Valid source file | `error_count == 0`, `parse_errors` empty |

### Helper Functions

```go
type ExpectedSymbol struct {
    Name       string
    Kind       string
    Exported   bool
    ParamCount int
    Receiver   *string
}

func assertSymbols(t *testing.T, got []Symbol, want []ExpectedSymbol) {
    t.Helper()
    for _, w := range want {
        found := false
        for _, g := range got {
            if g.Name == w.Name {
                found = true
                assert.Equal(t, w.Kind, g.Kind, "symbol %s kind", w.Name)
                assert.Equal(t, w.Exported, g.Exported, "symbol %s exported", w.Name)
                if w.ParamCount > 0 {
                    assert.Len(t, g.Params, w.ParamCount, "symbol %s params", w.Name)
                }
                if w.Receiver != nil {
                    assert.Equal(t, *w.Receiver, *g.Receiver, "symbol %s receiver", w.Name)
                }
                break
            }
        }
        assert.True(t, found, "expected symbol %s not found", w.Name)
    }
}
```

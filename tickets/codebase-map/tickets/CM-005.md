## Description

Implement the core Go binary that performs deterministic, language-agnostic symbol extraction using tree-sitter. This ticket covers the scaffolding, Go grammar support, and full extraction pipeline. Other languages (Python, TypeScript, R) are added in CM-006.

This is the foundation of the entire `/codebase-map` feature. Every other ticket depends on this binary producing correct, schema-compliant output.

```go
// ParseFile parses a single source file with tree-sitter
func ParseFile(source []byte, lang LanguageExtractor) (*sitter.Tree, error) {
    parser := sitter.NewParser()
    defer parser.Close()

    parser.SetLanguage(lang.TreeSitterLang())

    tree, err := parser.ParseCtx(context.Background(), nil, source)
    if err != nil {
        return nil, fmt.Errorf("parse %s: %w", lang.Language(), err)
    }

    return tree, nil
}

// DetectLanguage determines language from file extension
func DetectLanguage(path string) (LanguageExtractor, error) {
    ext := filepath.Ext(path)
    switch ext {
    case ".go":
        return &GoExtractor{}, nil
    case ".py":
        return &PythonExtractor{}, nil  // CM-006
    case ".ts", ".tsx":
        return &TypeScriptExtractor{}, nil  // CM-006
    case ".R", ".r":
        return &RExtractor{}, nil  // CM-006
    default:
        return nil, fmt.Errorf("unsupported extension: %s", ext)
    }
}
```

```go
func (g *GoExtractor) IsExported(name string) bool {
    if len(name) == 0 { return false }
    return unicode.IsUpper(rune(name[0]))
}

func (g *GoExtractor) ClassifyImport(importPath, projectModule string) string {
    // Read go.mod to get project module path
    // Internal: starts with projectModule
    // Stdlib: no dot in first path segment (e.g., "fmt", "os", "encoding/json")
    // External: everything else
    if strings.HasPrefix(importPath, projectModule) {
        return "internal"
    }
    firstSegment := strings.Split(importPath, "/")[0]
    if !strings.Contains(firstSegment, ".") {
        return "stdlib"
    }
    return "external"
}
```

#### 4. Go Queries (`queries/go.scm`)

```scheme
;; Function declarations
(function_declaration
  name: (identifier) @func.name
  parameters: (parameter_list) @func.params
  result: (_)? @func.returns
) @func.def

;; Method declarations (with receiver)
(method_declaration
  receiver: (parameter_list
    (parameter_declaration
      type: (_) @method.receiver_type))
  name: (field_identifier) @method.name
  parameters: (parameter_list) @method.params
  result: (_)? @method.returns
) @method.def

;; Type declarations (struct)
(type_declaration
  (type_spec
    name: (type_identifier) @type.name
    type: (struct_type) @type.body
  )
) @type.def

;; Interface declarations
(type_declaration
  (type_spec
    name: (type_identifier) @iface.name
    type: (interface_type) @iface.body
  )
) @iface.def

;; Const declarations
(const_declaration
  (const_spec
    name: (identifier) @const.name
    type: (_)? @const.type
    value: (_)? @const.value
  )
) @const.def

;; Var declarations
(var_declaration
  (var_spec
    name: (identifier) @var.name
    type: (_)? @var.type
  )
) @var.def

;; Import declarations
(import_declaration
  (import_spec
    path: (interpreted_string_literal) @import.path
  )
)
```

#### 5. Output Writer (`output.go`)

```go
// WriteModuleExtraction writes a single module's extraction to JSON
func WriteModuleExtraction(outputDir string, extraction *ModuleExtraction) error {
    filename := moduleToFilename(extraction.Module) + ".json"
    outPath := filepath.Join(outputDir, filename)

    data, err := json.MarshalIndent(extraction, "", "  ")
    if err != nil {
        return fmt.Errorf("marshal extraction: %w", err)
    }

    // Atomic write
    tmpPath := outPath + ".tmp"
    if err := os.WriteFile(tmpPath, data, 0644); err != nil {
        return fmt.Errorf("write tmp: %w", err)
    }
    return os.Rename(tmpPath, outPath)
}

// moduleToFilename converts "cmd/gogent-validate" to "cmd--gogent-validate"
func moduleToFilename(module string) string {
    return strings.ReplaceAll(module, "/", "--")
}
```

#### 6. Main Entry Point (`main.go`)

```go
func main() {
    // Parse flags
    rootPath := flag.String("path", ".", "Project root")
    outputDir := flag.String("output", ".claude/codebase-map/extract/", "Output directory")
    // ... other flags

    flag.Parse()

    // Ensure output directory exists
    os.MkdirAll(*outputDir, 0755)

    // Discover files
    moduleFiles, err := DiscoverFiles(*rootPath, opts)
    if err != nil {
        log.Fatalf("discovery: %v", err)
    }

    if len(moduleFiles) == 0 {
        fmt.Fprintln(os.Stderr, "No source files found")
        os.Exit(2)
    }

    // Extract each module
    var totalSymbols int
    for modulePath, files := range moduleFiles {
        extraction, err := ExtractModule(modulePath, files, *rootPath)
        if err != nil {
            log.Printf("[WARN] module %s: %v", modulePath, err)
            continue
        }

        if err := WriteModuleExtraction(*outputDir, extraction); err != nil {
            log.Printf("[ERROR] write %s: %v", modulePath, err)
            continue
        }

        totalSymbols += countSymbols(extraction)
        log.Printf("[OK] %s: %d files, %d symbols", modulePath, len(files), countSymbols(extraction))
    }

    // Update manifest
    if err := UpdateManifest(*rootPath, moduleFiles, *outputDir); err != nil {
        log.Printf("[WARN] manifest: %v", err)
    }

    // Summary to stderr
    fmt.Fprintf(os.Stderr, "Extracted %d modules, %d symbols\n", len(moduleFiles), totalSymbols)
}
```

## Code Guidelines

From go.md conventions:
1. **Error wrapping**: `fmt.Errorf("parse %s: %w", path, err)` with context
2. **Embed queries**: Use `//go:embed` for `.scm` query files
3. **Interface-driven**: `LanguageExtractor` interface for language extensibility
4. **Atomic writes**: Write-tmp-then-rename for output JSON
5. **Structured logging**: `log.Printf("[LEVEL] component: message")` pattern
6. **Flag parsing**: `flag` package for CLI, no external dependencies
7. **Parse error reporting**: After parsing, walk tree for ERROR-type nodes. Record each in `parse_errors` field. Log warning but continue extraction (tree-sitter is error-tolerant).
8. **Behavior contracts**: Code examples in this ticket describe WHAT, not HOW. The implementing agent writes code against the library selected in CM-001. Type definitions (output contracts) are concrete; parsing/query code is behavioral.

**CGo / Build System** (conditional on CM-001 outcome):
- If CGo selected: document `CGO_ENABLED=1` requirement, C compiler dependency, cross-compilation impact
- All grammars embedded in single binary for MVP. This is a known architectural trade-off (see AS1). Future option: WASM grammar loading for dynamic language support.

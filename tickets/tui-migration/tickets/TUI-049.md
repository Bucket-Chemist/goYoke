# TUI-049: Token usage progress bar

## Description

Add a visual progress bar to the status line showing context window utilization. Currently context percent is shown as text (`ctx:45.2%`). This ticket adds a compact bar visualization.

Add to `statusline.go` `View()` row 1, replacing the text-only `ctx` field:
```go
// renderContextBar returns a compact progress bar like: ctx:[=====     ] 52%
func (m StatusLineModel) renderContextBar() string
```

The bar is 10 characters wide (configurable). Fill character is `=` or Unicode block chars if available. Colors follow the same threshold logic as TUI-048: green < 50%, yellow < 80%, red >= 80%.

Implementation:
- Calculate fill count: `int(m.ContextPercent / 100 * barWidth)`
- Filled portion: theme color based on threshold
- Empty portion: muted style
- Format: `ctx:[filled+empty] N%`

When terminal width < 80 (narrow mode), fall back to text-only `ctx:N%` to save space.

This integrates with the existing `ContextPercent` field already populated from `ResultEvent.ModelUsage` in `app.go` lines 590-593.

# TUI-061: Tab highlight on focus change

## Description

Add a 300ms highlight flash on the active tab when focus changes. This provides immediate visual feedback to the user that the active tab/panel has changed.

Modify `internal/tui/components/tabbar/tabbar.go`:
1. Add `flashTab int` field ‚Äî the tab index currently flashing (-1 = none)
2. Add `flashStart time.Time` ‚Äî when the flash started
3. On focus change (`FocusChangedMsg` or equivalent), set `flashTab = newFocusIndex` and `flashStart = time.Now()`
4. In `View()`, render the flashing tab with an accent background for 300ms
5. After 300ms, `flashTab = -1` and re-render normally

Use tick-based animation (not a goroutine). The tab bar requests a tick every render cycle when `flashTab != -1`. On each tick, check if 300ms has elapsed. If yes, clear the flash.

Add `TabFlashMsg` to `messages.go`:
```go
type TabFlashMsg struct {
    TabIndex int
}
```

The flash does not affect the tab's focus state ‚Äî it is purely cosmetic.

# TUI-068: Dashboard metrics collapse/expand

## Description

Add collapsible section navigation to the dashboard panel. Currently the dashboard renders all metrics in a flat list. This ticket adds 4 collapsible sections with keyboard navigation following the proven agent tree pattern.

Modify `internal/tui/components/dashboard/dashboard.go` (or the dashboard-related component from TUI-032):

Add 4 collapsible sections:
1. **Session Overview** — session ID, duration, model, provider
2. **Cost & Tokens** — total cost, per-agent costs, token usage, context percent
3. **Agent Activity** — agent count, active agents, completed tasks, error count
4. **Performance** — NDJSON events/sec, modal latency, view render time

Each section has a toggle: Up/Down moves between sections, Enter collapses/expands. An expanded section shows its metrics; a collapsed section shows only its title and a summary metric.

Navigation follows the `settingstree` pattern from TUI-050:
- `cursor int` tracks selected section
- `expanded []bool` tracks expansion state per section

Use icons from `theme.Icons()` (TUI-045): expanded = `Icons.Running` (▶), collapsed = `Icons.Pending` (○) or equivalent.

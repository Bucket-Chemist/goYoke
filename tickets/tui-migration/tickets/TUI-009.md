# TUI-009: Banner, TabBar, StatusLine components

## Description

Implement the three fixed-height chrome components. (1) internal/tui/components/banner/banner.go: BannerModel with Init/Update/View. View() renders 'GOgent-Fortress' centered in a RoundedBorder box. Fixed height 3 rows. Full width. Uses theme.StyleTitle. (2) internal/tui/components/tabbar/tabbar.go: TabBarModel with tabs []TabDefinition, activeTab TabID. TabID is a string type. Predefined tabs: 'chat', 'agentConfig', 'teamConfig', 'telemetry'. View() renders horizontal tab strip with active tab highlighted (inverse style or underline). Fixed height 1 row. Tab switching via Alt+C/A/T/Y handled in Update(). (3) internal/tui/components/statusline/statusline.go: StatusLineModel with 8 data fields: SessionCost float64, TokenCount int, ContextPercent float64, PermissionMode string, ActiveModel string, Provider string, GitBranch string, AuthStatus string. View() renders a 2-row status bar with all fields formatted and laid out horizontally using Lipgloss. Each component must accept width int parameter for responsive sizing. All use value receivers per convention.

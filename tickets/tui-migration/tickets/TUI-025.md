# TUI-025: Toast notification system

## Description

Create the toast notification system in internal/tui/components/toast/toast.go. ToastModel: items []ToastItem, width int. ToastItem: Message string, Type string (info/success/warning/error), CreatedAt time.Time, Duration time.Duration (default 5s). Update() handles ToastMsg by appending new items. Uses tea.Every or tea.Tick to check for expired toasts every second and remove them. View() renders active toasts stacked at the top-right of the screen. Each toast is a small bordered box with type-specific coloring (error=red, success=green, warning=yellow, info=cyan). Maximum 3 visible toasts at once (oldest auto-dismissed if limit exceeded). Wire into root AppModel: overlay toast View() on top of the main layout using lipgloss.Place(). Toast appears for events like: context compaction, error notifications, CLI reconnection attempts.

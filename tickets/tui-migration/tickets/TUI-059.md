# TUI-059: Unified fuzzy search overlay

## Description

Create a unified search overlay that allows the user to search across conversation history, agent tree, and settings using a single fuzzy search interface. Activated via `ctrl+f` or `cmd+f`.

Create `internal/tui/components/search/` package:
```go
package search

// SearchResult is a single match returned by a SearchSource.
type SearchResult struct {
    Panel   string // "conversation", "agents", "settings"
    Label   string // display text for the result
    Score   int    // relevance score for sorting
    Action  func() // action to execute on selection (navigate to result)
}

// SearchOverlayModel is the floating search overlay.
type SearchOverlayModel struct {
    query    string
    results  []SearchResult
    selected int
    visible  bool
    width    int
    height   int
}
```

The `SearchSource` interface must be defined in `model/interfaces.go` (NOT in the `search/` package) to maintain the established import direction:
```go
// SearchSource is implemented by components that support content search.
// Defined in model/interfaces.go to avoid import cycles.
type SearchSource interface {
    Search(query string) []search.SearchResult
}
```

`ClaudePanelModel` and `AgentTreeModel` implement `SearchSource`. The search overlay queries all registered sources, merges results by score, and renders them in a floating overlay above the main content.

**Z-order:** Search overlay is dismissed when a modal opens. Modals > Search overlay > Toasts > Main content.

Add `searchOverlayWidget` interface to `model/interfaces.go`.

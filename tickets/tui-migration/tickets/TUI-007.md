# TUI-007: Keybinding registry

## Description

Create keybinding registry in internal/tui/config/keys.go using charmbracelet/bubbles/key package. Define a KeyMap struct with all keybindings from the migration plan Section 16. Global bindings (always active when no modal): ToggleFocus(Tab), CycleProvider(Shift+Tab), CycleRightPanel(Alt+R), CyclePermMode(Alt+P), Interrupt(Escape), ForceQuit(Ctrl+C), ClearScreen(Ctrl+L), ToggleTaskBoard(Alt+B). Tab shortcuts: TabChat(Alt+C), TabAgentConfig(Alt+A), TabTeamConfig(Alt+T), TabTelemetry(Alt+Y). Claude panel bindings: Submit(Enter), HistoryPrev(Up), HistoryNext(Down), ToggleToolExpansion(Alt+E), CycleExpansion(Alt+Shift+E). Agent panel bindings: AgentUp(Up), AgentDown(Down), AgentExpand(Enter). Modal bindings: ModalUp(Up), ModalDown(Down), ModalSelect(Enter), ModalCancel(Escape). Create a DefaultKeyMap() function that returns the fully configured KeyMap. Each binding should have appropriate Help() text for the bubbles/help component.

# TUI-060: Keyboard hint bar

## Description

Add a context-aware keyboard hint bar that displays relevant shortcuts based on the current focus state and active panel. The hint bar renders between the main content area and the status line.

Create `internal/tui/components/hintbar/` package:
```go
package hintbar

type HintSet struct {
    Label string   // e.g. "Tab: next panel"
    Key   string   // the key
    Desc  string   // description
}

type HintBarModel struct {
    hints  []HintSet
    width  int
    active bool
}

// SetContext updates the hint bar for the given focus context.
// Contexts: "main", "settings", "search", "modal", "plan"
func (m *HintBarModel) SetContext(context string)
```

Hint sets per context:
- `"main"`: Tab: next panel | Shift+Tab: prev panel | ctrl+f: search | /: slash cmd
- `"settings"`: Up/Down: navigate | Enter: toggle/select | Esc: close
- `"search"`: Up/Down: navigate | Enter: select | Esc: close
- `"modal"`: Enter: confirm | Esc: cancel
- `"plan"`: alt+v: view plan | Esc: exit plan

The hint bar renders as a single line of muted-style text, right-aligned or left-aligned depending on width.

Add `hintBarWidget` interface to `model/interfaces.go`.

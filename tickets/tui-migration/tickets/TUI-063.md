# TUI-063: Breadcrumb navigation trail

## Description

Add a breadcrumb navigation trail between the tab bar and the main content area. The breadcrumb shows the current location context: panel > sub-panel > current item.

Create `internal/tui/components/breadcrumb/` package:
```go
package breadcrumb

type BreadcrumbItem struct {
    Label string
    Key   string // keyboard shortcut to navigate to this level (optional)
}

type BreadcrumbModel struct {
    items []BreadcrumbItem
    width int
}

// SetCrumbs updates the breadcrumb trail.
func (m *BreadcrumbModel) SetCrumbs(items []BreadcrumbItem)
```

The breadcrumb renders as: `Claude > Conversation > [Tool: Read]` using the arrow icon from `theme.Icons().Arrow` (TUI-045). Items are separated by the arrow icon, styled with muted colors except the last item (current location) which is in normal style.

The breadcrumb is updated by `AppModel` when focus or panel context changes. Example breadcrumbs:
- Claude panel: `Claude > Conversation`
- Agent tree with selection: `Agents > Tree > go-tui`
- Settings with section: `Settings > Display`
- Plan preview: `Plan > Preview`

Add `breadcrumbWidget` interface to `model/interfaces.go`. Wire into `layout.go` row between tab bar and main content.

## Review Amendments

**m-3 (Minor — consider addressing):** Whether `breadcrumb/` warrants its own package vs a single file in an existing package. Decision: standalone package is appropriate because it has its own data model (`BreadcrumbItem` slice) and lifecycle (`SetCrumbs` updates). It is not a utility function.

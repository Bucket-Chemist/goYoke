# TUI-055: Task board enhancement

## Description

Upgrade the task board from a display-only component to an interactive component with status badges, progress summary, and filter shortcuts. Currently `taskboard.go` renders tasks as a flat list. This ticket adds cursor navigation, filter modes, and richer rendering.

Modifications to `internal/tui/components/taskboard/taskboard.go`:
1. Add cursor position (`cursor int`) for Up/Down navigation
2. Add filter mode (`filterMode TaskFilterMode`) with values: All, Running, Pending, Done
3. Filter shortcuts: `a` = All, `r` = Running, `p` = Pending, `d` = Done
4. Status badges: colored per task status using theme semantic styles from TUI-044
5. Progress summary in header: `Tasks: 3/10 running ŌĆó 5 pending ŌĆó 2 done`
6. Expanded detail on selected task: show tool calls, duration, agent type

The package documentation comment must be updated from "display-only" to "interactive".

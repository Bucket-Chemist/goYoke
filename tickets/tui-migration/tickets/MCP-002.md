# MCP-002: Subprocess spawner with stream-json parsing and SIGTERM/SIGKILL lifecycle

## Description

Implement the subprocess execution engine for `spawn_agent`. The spawner runs `claude -p` as a child process, streams its NDJSON output, and handles process lifecycle with graceful SIGTERM→SIGKILL escalation.

**Key deliverables:**
- `cmd/gofortress-mcp-standalone/spawner.go` — subprocess runner with NDJSON parsing
- `cmd/gofortress-mcp-standalone/spawner_test.go` — unit tests

**Subprocess contract:**
- Runs `claude --output-format stream-json -p <prompt>` with agent context
- Captures result, turn count, cost from final `result` event in stream
- Enforces configurable timeout (default 300s) with SIGTERM + 5s grace then SIGKILL
- Sets `GOGENT_NESTING_LEVEL`, `GOGENT_PARENT_AGENT`, `GOGENT_SPAWN_METHOD=mcp-cli` env vars

**NDJSON parsing:** Reads `{"type":"result","result":"...","total_cost_usd":0.0,"num_turns":N}` from stream. Handles truncated output via `is_error` flag.

**Nesting guard:** `validateNestingDepth()` reads `GOGENT_NESTING_LEVEL` and blocks at depth ≥ 5.

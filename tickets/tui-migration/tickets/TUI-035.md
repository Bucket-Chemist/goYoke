# TUI-035: Clipboard, search, and input history

## Description

Implement remaining UX features. (1) Clipboard support in internal/tui/util/clipboard.go: CopyToClipboard(text string) error using github.com/atotto/clipboard (already an indirect dependency). Add a keybinding (e.g., Alt+Y in agents panel to copy agent output). (2) Search within conversation in internal/tui/components/claude/search.go: SearchModel with query textinput, results []int (matching message indices), currentResult int. Triggered by '/' in the Claude panel (when focused). Highlights matching text in messages. Ctrl+N/Ctrl+P to jump between results. Escape to dismiss search. (3) Input history in internal/tui/components/claude/history.go: InputHistory struct with entries []string, maxSize int, index int. Methods: Add(entry), Previous() string, Next() string, Reset(). Persist history to session directory (inputhistory.json). Load on startup. Wire into ClaudePanelModel: Up arrow loads previous history entry into input, Down arrow loads next. Enter adds current input to history before submitting.

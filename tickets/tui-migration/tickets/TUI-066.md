# TUI-066: Rich modal styling

## Description

Upgrade the modal system with type-specific border colors, header icons, visual shadow effects, and radio button-style option selection. Currently all modals use the same basic border style.

Modify `internal/tui/components/modals/model.go`:

1. **Type-specific border colors:** Each `ModalType` gets a border color from the theme semantic system:
   - Permission modals → `theme.WarningStyle()` (yellow border)
   - Error/failure modals → `theme.ErrorStyle()` (red border)
   - Info/confirmation modals → `theme.InfoStyle()` (cyan border)
   - Destructive action modals → `theme.DangerStyle()` (red + bold border)

2. **Header icons:** Modal header includes the type-appropriate icon from `theme.Icons()`:
   - Permission: `Icons.Warning`
   - Error: `Icons.Error`
   - Info: `Icons.Info`
   - Success: `Icons.Success`

3. **Shadow effect:** Add a 1-character shadow below/right of the modal border using a darker background block character for depth illusion.

4. **Radio button options:** Replace the current `[1]` `[2]` numbering with `( )` / `(*)` radio button style for option lists.

# TUI-020: Agent tree view component

## Description

Create the agent tree view component in internal/tui/components/agents/tree.go. AgentTreeModel: treeNodes []AgentTreeNode (cached from AgentRegistry.Tree()), selectedIdx int, focused bool, width int, height int. View() renders the tree with indentation based on depth, status icons from theme (IconRunning, IconComplete, etc.), agent type, truncated description, and activity preview. Selected agent is highlighted. Tree lines use Unicode box-drawing characters. Format per line: '{indent}{connector}{icon} {type}: {description} [{activity}]'. Connector characters: last child uses 'L', others use 'T', with vertical lines for parent depth. Update() handles Up/Down for selection when focused. On selection change, emit AgentSelectedMsg with the agent ID. Create internal/tui/components/agents/detail.go for AgentDetailModel: shows full detail of the selected agent. Fields displayed: Status (with color), Model, Tier, Duration, Cost, Token count, Live activity line, Error output (if error status). Both components accept width and height for responsive sizing.

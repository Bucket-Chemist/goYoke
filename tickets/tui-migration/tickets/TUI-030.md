# TUI-030: Provider tab bar UI (P7-7c)

## Description

Create the provider tab bar UI component in internal/tui/components/providers/tabbar.go. ProviderTabBarModel: providers []ProviderID, active ProviderID, width int. View() renders horizontal tab strip: Anthropic | Google | OpenAI | Local with the active provider highlighted (bold + underline or inverse). Tabs are styled with provider-specific colors if desired, or neutral with active highlighting. The tab bar sits above the main content area, below the top TabBar. Update() handles Shift+Tab to cycle providers, emits ProviderSwitchMsg. Wire into root AppModel View(): render provider tab bar between the top TabBar and the main content area when more than one provider is configured. Hide when only Anthropic is configured (default single-provider mode).

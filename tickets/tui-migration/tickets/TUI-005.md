# TUI-005: Theme system and Lipgloss styles

## Description

Create the theme system for the Go TUI by translating theme.ts to Go Lipgloss styles. Create file internal/tui/config/theme.go. Define package-level color constants using lipgloss.AdaptiveColor for light/dark terminal support. Use ANSI color numbers (0-15) for base theme to respect user terminal colorscheme, reserve hex codes for accent colors. Define style variables: StyleFocusedBorder (RoundedBorder, primary foreground), StyleUnfocusedBorder (NormalBorder, muted foreground), StyleStatusBar, StyleTitle, StyleSubtle, StyleHighlight, StyleError, StyleSuccess, StyleWarning, StyleMuted. Map all colors, borders, and icons from the existing theme.ts: ColorPrimary=6(cyan), ColorSecondary=4(blue), ColorAccent=5(magenta), ColorSuccess=2(green), ColorWarning=3(yellow), ColorError=1(red), ColorMuted=8(gray). Define icon constants: IconRunning='>', IconComplete='*', IconError='!', IconPending='.', etc. Export a Theme struct that bundles all styles for testability and potential theme switching.

# TUI-006: Focus management and FocusTarget

## Description

Create focus management system in internal/tui/model/focus.go. Define FocusTarget type as int with iota constants: FocusClaude, FocusAgents. Implement FocusNext() and FocusPrev() methods that cycle between targets. The focus state is stored on the root AppModel.focus field. When focus changes: (1) the previously focused panel's border switches to StyleUnfocusedBorder, (2) the newly focused panel's border switches to StyleFocusedBorder, (3) only the focused panel receives tea.KeyMsg in Update(). Tab key triggers focus cycling. The FocusTarget determines which child model receives keyboard input in the root Update() method. Also define RightPanelMode type with iota: RPMAgents, RPMDashboard, RPMSettings, RPMTelemetry. Alt+R cycles through these modes.

# TUI-032: Dashboard, Settings, Telemetry, PlanPreview, TaskBoard panels

## Description

Create the remaining right-panel views. (1) internal/tui/components/dashboard/dashboard.go: DashboardModel shows session summary stats: total cost, total tokens, messages sent, agents spawned, teams run, session duration. Simple key-value display. (2) internal/tui/components/settings/settings.go: SettingsModel displays current configuration: model, permission mode, provider, MCP servers, session directory. Read-only display. (3) internal/tui/components/telemetry/telemetry.go: TelemetryModel shows routing decisions (from $XDG_DATA_HOME/gogent/routing-decisions.jsonl), handoffs, sharp edges. Uses a viewport for scrolling. Loads data via tea.Cmd (file read). (4) internal/tui/components/planpreview/planpreview.go: PlanPreviewModel shows plan markdown (from ExitPlanMode flow). Uses Glamour for rendering. Content set via PlanPreviewContentMsg. (5) internal/tui/components/taskboard/taskboard.go: TaskBoardModel shows active/completed tasks in a compact strip. Alt+B toggles between Active and Done views. Collapsible (0 or 8 rows). All panels accept width and height for responsive sizing. Wire into root AppModel: right panel renders based on rightPanelMode. Alt+R cycles the mode.

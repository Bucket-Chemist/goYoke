# TUI-052: Shift+Tab reverse navigation

## Description

Rebind Shift+Tab from `CycleProvider` to `ReverseToggleFocus` (reverse focus cycling). This is a standard keyboard convention: Tab moves focus forward, Shift+Tab moves focus backward.

Changes required in `config/keys.go`:
1. Add `ReverseToggleFocus` binding to the `Global` key group, mapped to `"shift+tab"`
2. Rebind `CycleProvider` from `"shift+tab"` to `"alt+shift+p"` (or equivalent non-conflicting binding)
3. Alt+P continues to cycle permission modes (no change)

Changes required in `internal/tui/model/key_handlers.go` (created by TUI-043):
1. Add handling for `ReverseToggleFocus` key — calls `m.FocusPrev()` (which already exists at `focus.go:43`)
2. Handle the new `CycleProvider` binding in the appropriate case

**This is a breaking keybinding change.** The existing `shift+tab` behavior (cycle provider) moves to `alt+shift+p`. Document the change clearly in code comments and ensure existing users can discover the new binding via the keyboard hint bar (TUI-060).

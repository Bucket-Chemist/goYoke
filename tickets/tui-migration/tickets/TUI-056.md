# TUI-056: Plan preview modal with Glamour

## Description

Add a full-screen plan preview modal that renders Markdown plan content using Glamour. When a plan is active (`rightPanelMode == RPMPlanPreview`), the user can press `alt+v` to open a scrollable full-screen modal showing the plan with rich Markdown rendering.

Create `internal/tui/components/modals/plan_modal.go`:
```go
package modals

// PlanViewModal is a full-screen Glamour-rendered plan viewer.
type PlanViewModal struct {
    content   string    // raw markdown content
    rendered  string    // pre-rendered Glamour output
    viewport  viewport.Model
    width     int
    height    int
}

func NewPlanViewModal(content string) PlanViewModal
func (m PlanViewModal) Init() tea.Cmd
func (m PlanViewModal) Update(msg tea.Msg) (PlanViewModal, tea.Cmd)
func (m PlanViewModal) View() string
```

Add `PlanView` to the `ModalType` enum in `modals/types.go`. The modal is activated via `alt+v` when `rightPanelMode == RPMPlanPreview` (as defined in `model/focus.go`).

**Trigger condition:** `alt+v` activates the plan modal when `rightPanelMode == RPMPlanPreview`. If plan preview is not active, `alt+v` is ignored.

Glamour rendering is pre-computed in `SetContent()` (not in `View()`), using the existing `util.RenderMarkdown()` from `internal/tui/util/markdown.go` (TUI-023). The rendered string is stored and scrolled via a `viewport.Model`.

Scrolling: Up/Down/PgUp/PgDn scroll the viewport. `Esc` or `q` closes the modal.

# TUI-054: Slash command execution via CLI

## Description

Wire the slash command dropdown (TUI-053) into the Claude panel's input handling. When a slash command is selected via the dropdown or typed directly, route it appropriately.

Two classes of slash commands:
1. **Local commands** (handled in TUI): `/clear`, `/help` тАФ processed directly in `ClaudePanelModel.Update()` without passing to CLI
2. **Remote commands** (pass to CLI): all others тАФ sent to the Claude CLI subprocess via the existing `SendMessage` interface

Modify `internal/tui/components/claude/panel.go`:
1. Detect input starting with `/` on Enter press
2. Show `SlashCmdModel` dropdown while typing
3. On selection (Enter or Tab-complete):
   - For `/clear`: clear conversation history locally
   - For `/help`: show a help modal or local message
   - For all others: call `m.sendMessage(input)` which passes to CLI subprocess

Add `SlashExecutedMsg` to `messages.go`:
```go
type SlashExecutedMsg struct {
    Command string
    Args    string
    IsLocal bool
}
```

The CLI subprocess receives remote slash commands as plain text messages. The Claude CLI routes them through its skill system natively.

# TUI-033: Session persistence (load/save)

## Description

Implement session persistence in internal/tui/session/persistence.go. SessionData struct: ID string, Name string, CreatedAt time.Time, LastUsed time.Time, Cost float64, ToolCalls int, ProviderSessionIDs map[ProviderID]string, ProviderModels map[ProviderID]string, ActiveProvider ProviderID. Functions: LoadSession(id string) (*SessionData, error) -- loads from ~/.claude/sessions/{id}/session.json. SaveSession(data *SessionData) error -- writes to ~/.claude/sessions/{id}/session.json with atomic write (write to temp file, rename). SetupSessionDir(sessionID string) (string, error) -- creates session directory if not exists, writes .claude/current-session marker, creates .claude/tmp symlink pointing to session dir. NewSessionID() string -- generates YYYYMMDD.UUID format. Also create internal/tui/session/history.go: LoadConversationHistory(sessionID string, provider ProviderID) ([]DisplayMessage, error) and SaveConversationHistory(sessionID, provider, messages). Wire into root AppModel: Init() loads session if --session-id provided. Auto-save triggered on cost changes (debounced with 5s cooldown via tea.Every). Save on Ctrl+C/SIGTERM shutdown.

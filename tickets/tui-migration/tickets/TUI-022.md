# TUI-022: Claude conversation panel

## Description

Create the Claude conversation panel in internal/tui/components/claude/panel.go. ClaudePanelModel: messages []DisplayMessage, viewport viewport.Model, input textinput.Model, inputHistory []string, historyIdx int, streaming bool, autoScroll bool, width int, height int, focused bool. DisplayMessage struct: Role string (user/assistant/system), Content string (rendered), ToolBlocks []ToolBlock (collapsed/expanded), Timestamp time.Time. The panel has two areas: (1) conversation viewport (scrollable, takes remaining height), (2) input area at bottom (textinput with prompt indicator). In Update(): handle AssistantMsg by appending/updating display messages. Handle StreamEventMsg by updating the last message incrementally (for token streaming). Handle ResultMsg by finalizing the current response. Handle Enter key by submitting input text via CLIDriver.SendMessage(). Handle Up/Down for input history navigation. Auto-scroll: viewport.GotoBottom() when autoScroll is true. Disable autoScroll when user scrolls up (viewport.AtBottom() becomes false). Re-enable autoScroll on new content when at bottom. View(): render conversation messages with role indicators (You: / Claude:), then the input line. Markdown rendering of assistant messages is deferred to Phase 6 -- for now use plain text.

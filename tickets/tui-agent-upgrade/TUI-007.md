# TUI-007: Add Standard tier tab-toggle (Alt+H) for health view

## Description

Add `Alt+H` keybinding that toggles `showHealthInAgents` in RPMAgents mode, allowing Standard tier users to switch between agent detail and health dashboard. Include a tab indicator showing which view is active.

## Acceptance Criteria

- [ ] `Alt+H` toggles `showHealthInAgents` when `rightPanelMode == RPMAgents`
- [ ] Tab indicator renders at Standard tier: `[Agents]  Health` or `Agents  [Health]` with active tab highlighted
- [ ] Toggle resets to `false` when switching panel modes (Tab key)
- [ ] No effect at Wide/Ultra (50/50 split always shown)
- [ ] `go build ./...` passes

## Files

- `internal/tui/model/key_handlers.go` — Alt+H handler + reset on mode change (~15 lines)
- `internal/tui/model/layout.go` — Tab indicator rendering at Standard tier (~10 lines)

## Context

### Tab indicator visual:
```
Active agents:  [Agents]  Health     (Agents bold/highlighted, Health muted)
Active health:   Agents  [Health]    (Health bold/highlighted, Agents muted)
```

Render the indicator as the first line of the right panel content area at Standard tier only. Use `config.StyleFocused` for active tab and `config.StyleMuted` for inactive.

### Reset behavior:
When `NextRightPanelMode()` is called (Tab key cycling through panel modes), reset `showHealthInAgents = false` so the user always returns to the default agents view.

# TUI-005: Implement TeamsHealthModel component

## Description

Build the teams health dashboard as a new Bubbletea component. This is a display-only panel that shows the active team's health at a glance: header, budget bar, wave-grouped members with health indicators, and a summary footer.

**Auto-scrolls to the current (most active) wave** when content exceeds available height. No keyboard scrolling — this is a passive monitoring panel.

## Acceptance Criteria

- [ ] `TeamsHealthModel` implements `teamsHealthWidget` interface: `View()`, `SetSize(w, h)`, `SetTier(LayoutTier)`
- [ ] Shows team header: name, status, PID, uptime
- [ ] Budget bar with Unicode blocks (`█░`) and color thresholds: green <70%, yellow 70-90%, red >90%
- [ ] Members grouped by wave with wave headers

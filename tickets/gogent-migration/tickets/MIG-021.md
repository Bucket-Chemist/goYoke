# MIG-021: CI grep check for residual .claude/ write paths

## Description

Create a CI check script at scripts/check-claude-writes.sh that:

1. Greps all .go files (excluding _test.go, vendor/, .claude/) for patterns: os.WriteFile + .claude, os.MkdirAll + .claude, os.OpenFile + .claude, os.Create + .claude
2. Excludes known legitimate config reads (.claude/agents/, .claude/routing-schema.json, .claude/conventions/, .claude/rules/, .claude/CLAUDE.md)
3. Exits with non-zero status if any matches found
4. Outputs clear error message listing offending files

Add a Makefile target or document how to run: make check-claude-writes

This prevents regression — any new code that writes to .claude/ will fail CI.

## Acceptance Criteria

- [ ] Script exits 0 when no .claude/ write paths found
- [ ] Script exits 1 with clear output when .claude/ writes detected
- [ ] Config reads excluded from check
- [ ] Test files excluded from check

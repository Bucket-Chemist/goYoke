# TUI-001: SDK API Verification

## Description

Verify that the `@anthropic-ai/claude-agent-sdk` exports the APIs assumed in the conversion plan. This is a GATE ticket - failure blocks all subsequent work.

## Acceptance Criteria

- [ ] SDK installed successfully in test project
- [ ] `query()` function exists and is callable
- [ ] `createSdkMcpServer()` function exists
- [ ] `tool()` function exists for defining MCP tools
- [ ] MCP tool invocation works with simple handler
- [ ] Streaming events from `query()` work as expected

## Tasks

1. Create minimal test project:
   ```bash
   mkdir /tmp/sdk-test && cd /tmp/sdk-test
   npm init -y
   npm install @anthropic-ai/claude-agent-sdk typescript tsx
   ```

2. Write `test-sdk.ts`:
   ```typescript
   import { query, createSdkMcpServer, tool } from "@anthropic-ai/claude-agent-sdk";
   import { z } from "zod";

   // Verify exports exist
   console.log("query:", typeof query);
   console.log("createSdkMcpServer:", typeof createSdkMcpServer);
   console.log("tool:", typeof tool);

   // Create test MCP server
   const testServer = createSdkMcpServer({
     name: "test-server",
     version: "1.0.0",
     tools: [
       tool(
         "echo",
         "Echo input back",
         { message: z.string() },
         async (args) => ({ content: [{ type: "text", text: args.message }] })
       )
     ]
   });

   console.log("MCP server created:", testServer);
   ```

3. Execute and verify:
   ```bash
   npx tsx test-sdk.ts
   ```

4. Document any API differences from plan assumptions

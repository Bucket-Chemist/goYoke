### Part 2: Hook Integration Test

1. Create minimal Node.js script that spawns Claude CLI:
   ```typescript
   // test-hooks.ts
   import { spawn } from "child_process";

   const claude = spawn("claude", ["--print-session-id"], {
     stdio: ["pipe", "pipe", "pipe"],
     env: { ...process.env }
   });

   claude.stdout.on("data", (data) => {
     console.log("stdout:", data.toString());
   });

   claude.on("close", (code) => {
     console.log("Exit code:", code);
   });
   ```

2. Verify hooks fire:
   - `gogent-load-context` (SessionStart)
   - `gogent-validate` (PreToolUse)
   - `gogent-sharp-edge` (PostToolUse)

3. Check hook output files are written correctly

# TUI-011: MCP Server with Tools

## Description

Implement the in-process MCP server using the Claude Agent SDK with all four interactive tools (ask_user, confirm_action, request_input, select_option).

## Acceptance Criteria

- [ ] MCP server created with `createSdkMcpServer()`
- [ ] `ask_user` tool implemented
- [ ] `confirm_action` tool implemented
- [ ] `request_input` tool implemented
- [ ] `select_option` tool implemented
- [ ] All tools integrate with modal queue
- [ ] Tools return correct response format
- [ ] Unit tests pass

## Tasks

1. Create `src/mcp/server.ts`:
   ```typescript
   import { createSdkMcpServer, tool } from "@anthropic-ai/claude-agent-sdk";
   import { z } from "zod";
   import { useStore } from "../store/index.js";

   export const mcpServer = createSdkMcpServer({
     name: "gofortress-interactive",
     version: "1.0.0",
     tools: [
       // Tools defined below
     ]
   });
   ```

2. Implement `ask_user` tool (`src/mcp/tools/askUser.ts`):
   ```typescript
   export const askUserTool = tool(
     "ask_user",
     "Ask the user a question with optional predefined options",
     {
       message: z.string().describe("The question to ask"),
       options: z.array(z.string()).optional().describe("Predefined answer options"),
       default: z.string().optional().describe("Default value if timeout")
     },
     async (args) => {
       const response = await useStore.getState().enqueue({
         type: "ask",
         payload: {
           message: args.message,
           options: args.options,
           defaultValue: args.default
         }
       });
       return { content: [{ type: "text", text: response.value }] };
     }
   );
   ```

3. Implement `confirm_action` tool (`src/mcp/tools/confirmAction.ts`):
   ```typescript
   export const confirmActionTool = tool(
     "confirm_action",
     "Request user confirmation for an action",
     {
       action: z.string().describe("Description of the action to confirm"),
       destructive: z.boolean().optional().describe("Whether action is destructive")
     },
     async (args) => {
       const response = await useStore.getState().enqueue({
         type: "confirm",
         payload: {
           action: args.action,
           destructive: args.destructive ?? false
         }
       });
       return {
         content: [{
           type: "text",
           text: JSON.stringify({
             confirmed: response.confirmed,
             cancelled: response.cancelled
           })
         }]
       };
     }
   );
   ```

4. Implement `request_input` tool (`src/mcp/tools/requestInput.ts`)

5. Implement `select_option` tool (`src/mcp/tools/selectOption.ts`)

6. Write unit tests for each tool

# TUI-007: Layout & Banner Components

## Description

Implement the main Layout component with 2-panel split and the Banner component showing session info, cost, and status.

## Acceptance Criteria

- [ ] Layout renders 2-panel split (70/30)
- [ ] Tab switches focus between panels
- [ ] Escape exits (when no modal active)
- [ ] Focus state visually indicated (border color)
- [ ] Banner shows session ID
- [ ] Banner shows cumulative cost
- [ ] Banner shows current status
- [ ] Components responsive to terminal resize
- [ ] **Uses theme constants from `src/config/theme.ts` (see TUI-003)**

## Tasks

### Layout Component

1. Create `src/components/Layout.tsx`:
   ```typescript
   import { Box, useInput } from "ink";
   import { useStore } from "../store/index.js";
   import { Banner } from "./Banner.js";

   export function Layout(): JSX.Element {
     const { focusedPanel, setFocusedPanel, modalQueue } = useStore();

     useInput((input, key) => {
       if (key.tab) {
         setFocusedPanel(focusedPanel === "claude" ? "agents" : "claude");
       }
       if (key.escape && modalQueue.length === 0) {
         process.exit(0);
       }
     });

     return (
       <Box flexDirection="column" height="100%">
         <Banner />
         <Box flexDirection="row" flexGrow={1}>
           {/* TODO: Import from theme.ts - colors.focused, colors.unfocused, etc. */}
           <Box width="70%" borderStyle="single" borderColor={focusedPanel === "claude" ? "cyan" : "gray"}>
             {/* ClaudePanel placeholder */}
           </Box>
           <Box width="30%" flexDirection="column">
             <Box height="60%" borderStyle="single" borderColor={focusedPanel === "agents" ? "cyan" : "gray"}>
               {/* AgentTree placeholder */}
             </Box>
             <Box height="40%" borderStyle="single">
               {/* AgentDetail placeholder */}
             </Box>
           </Box>
         </Box>
       </Box>
     );
   }
   ```

### Banner Component

2. Create `src/components/Banner.tsx`:
   ```typescript
   import { Box, Text } from "ink";
   import { useStore } from "../store/index.js";

   export function Banner(): JSX.Element {
     const { sessionId, totalCost, streaming } = useStore();

     const status = streaming ? "Streaming..." : "Ready";
     const costDisplay = `$${totalCost.toFixed(4)}`;

     return (
       <Box
         borderStyle="round"
         {/* TODO: Import from theme.ts - colors.focused, colors.unfocused, etc. */}
         borderColor="cyan"
         paddingX={1}
         justifyContent="space-between"
       >
         <Text bold color="cyan">GOfortress</Text>
         <Text>Session: {sessionId ?? "None"}</Text>
         <Text>Cost: {costDisplay}</Text>
         <Text color={streaming ? "yellow" : "green"}>{status}</Text>
       </Box>
     );
   }
   ```

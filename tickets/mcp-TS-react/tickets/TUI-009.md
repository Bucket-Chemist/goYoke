# TUI-009: Modal System Components

## Description

Implement the modal overlay system and all modal type components (Ask, Confirm, Input, Select) that integrate with the modal queue.

## Acceptance Criteria

- [ ] ModalOverlay renders centered over content
- [ ] Modal captures keyboard focus
- [ ] Escape cancels modal
- [ ] Enter submits modal (context-dependent)
- [ ] AskModal: displays question with optional buttons
- [ ] ConfirmModal: Yes/No with destructive styling
- [ ] InputModal: text input with placeholder
- [ ] SelectModal: arrow-navigable list
- [ ] All modals resolve their Promise on completion

## Tasks

### Modal Overlay

1. Create `src/components/Modal.tsx`:
   ```typescript
   interface ModalOverlayProps {
     request: ModalRequest;
   }

   export function ModalOverlay({ request }: ModalOverlayProps): JSX.Element {
     const { dequeue, cancel } = useStore();

     useInput((input, key) => {
       if (key.escape) {
         cancel(request.id);
       }
     });

     return (
       <Box
         position="absolute"
         top={0}
         left={0}
         width="100%"
         height="100%"
         justifyContent="center"
         alignItems="center"
       >
         <Box
           borderStyle="double"
           borderColor="yellow"
           padding={2}
           flexDirection="column"
         >
           <CurrentModal request={request} onComplete={(response) => dequeue(request.id, response)} />
         </Box>
       </Box>
     );
   }
   ```

### Individual Modals

2. Create `src/components/modals/AskModal.tsx`:
   - Displays message text
   - Optional button options (arrow key navigation)
   - Free text input if no options
   - Enter submits

3. Create `src/components/modals/ConfirmModal.tsx`:
   - Displays action description
   - Yes/No buttons
   - Destructive actions highlighted in red
   - Y/N keyboard shortcuts

4. Create `src/components/modals/InputModal.tsx`:
   - Displays prompt
   - Text input with placeholder
   - Enter submits

5. Create `src/components/modals/SelectModal.tsx`:
   - Displays message
   - List of options (arrow navigation)
   - Enter selects current option
   - Number keys for quick select

### Integration

6. Update Layout to render modal when queue non-empty:
   ```typescript
   {modalQueue.length > 0 && <ModalOverlay request={modalQueue[0]} />}
   ```

7. Write component tests for each modal type

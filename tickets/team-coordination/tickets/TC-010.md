# TC-010: Rewrite Inter-Wave Script as Go Binary

**Phase**: Phase 2: Go Binary Implementation
**Priority**: HIGH
**Blocked By**: TC-008 (needs the team-run binary to exist for integration)
**Effort**: 1-2 days

## Description

Rewrite `gogent-team-prepare-synthesis.sh` as `cmd/gogent-team-prepare-synthesis/main.go`. This eliminates the `jq` dependency and provides typed JSON parsing with proper error handling. The binary runs between Wave 1 (Einstein + Staff-Architect) and Wave 2 (Beethoven) to prepare a curated synthesis input file.

### CLI Interface

**Usage**:
```bash
gogent-team-prepare-synthesis <team-directory>
```

**Behavior**:
- Reads `stdout_einstein.json` and `stdout_staff-arch.json` from team directory
- Extracts key sections from each analyst's output
- Writes `pre-synthesis.md` to team directory
- Exits 0 on success, 1 on fatal error (team directory not found)
- Logs warnings to stderr for missing/malformed files but continues

// loadAndExtract loads a JSON file with graceful degradation
func loadAndExtract(path string, target interface{}) error {
    data, err := os.ReadFile(path)
    if err != nil {
        return fmt.Errorf("read file: %w", err)
    }

    if err := json.Unmarshal(data, target); err != nil {
        return fmt.Errorf("parse JSON: %w", err)
    }

    return nil
}

// extractEinsteinSections extracts key sections from Einstein's output with fallbacks
func extractEinsteinSections(teamDir string) EinsteinSections {
    path := filepath.Join(teamDir, "stdout_einstein.json")

    var output EinsteinOutput
    if err := loadAndExtract(path, &output); err != nil {
        log.Printf("[WARN] Failed to load Einstein output: %v", err)
        return EinsteinSections{
            RootCause:             "(Einstein output unavailable)",
            ChallengedAssumptions: []string{"(fallback: could not parse Einstein output)"},
            NovelApproaches:       []string{},
            CriticalFailureModes:  []string{},
            OpenQuestions:         []string{},
        }
    }

    // Status check
    if output.Status != "completed" {
        log.Printf("[WARN] Einstein status: %s", output.Status)
    }

    return EinsteinSections{
        RootCause:             output.Content.RootCause,
        ChallengedAssumptions: output.Content.ChallengedAssumptions,
        NovelApproaches:       output.Content.NovelApproaches,
        CriticalFailureModes:  output.Content.CriticalFailureModes,
        OpenQuestions:         output.Content.OpenQuestions,
    }
}

// extractStaffArchSections extracts key sections from Staff-Architect's output with fallbacks
func extractStaffArchSections(teamDir string) StaffArchSections {
    path := filepath.Join(teamDir, "stdout_staff-arch.json")

    var output StaffArchOutput
    if err := loadAndExtract(path, &output); err != nil {
        log.Printf("[WARN] Failed to load Staff-Architect output: %v", err)
        return StaffArchSections{
            ArchitectureSmells:  []string{"(fallback: could not parse Staff-Architect output)"},
            ContractorReadiness: "unknown",
            RefinementGuidance:  "",
            CodeReviewFindings:  []string{},
        }
    }

    if output.Status != "completed" {
        log.Printf("[WARN] Staff-Architect status: %s", output.Status)
    }

    return StaffArchSections{
        ArchitectureSmells:  output.Content.ArchitectureSmells,
        ContractorReadiness: output.Content.ContractorReadiness,
        RefinementGuidance:  output.Content.RefinementGuidance,
        CodeReviewFindings:  output.Content.CodeReviewFindings,
    }
}

### Markdown Formatting (`markdown.go`)

```
func generateMarkdown(einstein EinsteinSections, staffArch StaffArchSections) string {
    var buf strings.Builder

    buf.WriteString("# Pre-Synthesis: Analyst Findings\n\n")
    buf.WriteString("**Generated by**: gogent-team-prepare-synthesis\n")
    buf.WriteString(fmt.Sprintf("**Timestamp**: %s\n\n", time.Now().Format(time.RFC3339)))

    buf.WriteString("---\n\n")

    // Einstein sections
    buf.WriteString("## Einstein: Theoretical Analysis\n\n")

    buf.WriteString("### Root Cause\n\n")
    buf.WriteString(formatParagraph(einstein.RootCause))
    buf.WriteString("\n\n")

    buf.WriteString("### Challenged Assumptions\n\n")
    buf.WriteString(formatList(einstein.ChallengedAssumptions))
    buf.WriteString("\n\n")

    buf.WriteString("### Novel Approaches\n\n")
    buf.WriteString(formatList(einstein.NovelApproaches))
    buf.WriteString("\n\n")

    buf.WriteString("### Critical Failure Modes\n\n")
    buf.WriteString(formatList(einstein.CriticalFailureModes))
    buf.WriteString("\n\n")

    buf.WriteString("### Open Questions\n\n")
    buf.WriteString(formatList(einstein.OpenQuestions))
    buf.WriteString("\n\n")

    buf.WriteString("---\n\n")

    // Staff-Architect sections
    buf.WriteString("## Staff-Architect: Critical Review\n\n")

    buf.WriteString("### Architecture Smells\n\n")
    buf.WriteString(formatList(staffArch.ArchitectureSmells))
    buf.WriteString("\n\n")

    buf.WriteString("### Contractor Readiness\n\n")
    buf.WriteString(fmt.Sprintf("**Status**: %s\n\n", staffArch.ContractorReadiness))
    if staffArch.RefinementGuidance != "" {
        buf.WriteString("**Refinement Guidance**:\n\n")
        buf.WriteString(formatParagraph(staffArch.RefinementGuidance))
        buf.WriteString("\n\n")
    }

    buf.WriteString("### Code Review Findings\n\n")
    buf.WriteString(formatList(staffArch.CodeReviewFindings))
    buf.WriteString("\n\n")

    buf.WriteString("---\n\n")
    buf.WriteString("**Next Step**: Beethoven synthesizes these findings into unified recommendations.\n")

    return buf.String()
}

func formatParagraph(text string) string {
    if text == "" {
        return "(No content provided)"
    }
    return text
}

func formatList(items []string) string {
    if len(items) == 0 {
        return "(None identified)"
    }

    var buf strings.Builder
    for _, item := range items {
        buf.WriteString(fmt.Sprintf("- %s\n", item))
    }
    return buf.String()
}
```

### Main Entry Point (`main.go`)

```
func main() {
    if len(os.Args) < 2 {
        fmt.Fprintln(os.Stderr, "Usage: gogent-team-prepare-synthesis <team-directory>")
        os.Exit(1)
    }

    teamDir := os.Args[1]

    // Verify team directory exists
    if _, err := os.Stat(teamDir); os.IsNotExist(err) {
        fmt.Fprintf(os.Stderr, "Error: Team directory does not exist: %s\n", teamDir)
        os.Exit(1)
    }

    // Extract sections from both analysts
    einstein := extractEinsteinSections(teamDir)
    staffArch := extractStaffArchSections(teamDir)

    // Generate markdown
    markdown := generateMarkdown(einstein, staffArch)

    // Write to pre-synthesis.md
    outputPath := filepath.Join(teamDir, "pre-synthesis.md")
    if err := os.WriteFile(outputPath, []byte(markdown), 0644); err != nil {
        fmt.Fprintf(os.Stderr, "Error: Failed to write pre-synthesis.md: %v\n", err)
        os.Exit(1)
    }

    fmt.Printf("Pre-synthesis markdown written to %s\n", outputPath)
}
```

### Graceful Degradation Pattern

**Missing file scenario**:
```
stdout_einstein.json not found
→ extractEinsteinSections() returns fallback struct
→ Markdown contains "(Einstein output unavailable)"
→ Beethoven receives partial synthesis input
→ Binary exits 0 (not fatal)
```

**Malformed JSON scenario**:
```
stdout_staff-arch.json contains invalid JSON
→ json.Unmarshal() returns error
→ extractStaffArchSections() returns fallback struct
→ Markdown contains "(fallback: could not parse...)"
→ Binary exits 0 (not fatal)
```

**Both files missing scenario**:
```
Both stdout files missing
→ Both extractors return fallback structs
→ Markdown contains all fallback text
→ Beethoven receives degraded input (better than crashing)
→ Binary exits 0
```

## Code Guidelines

From go.md conventions:

1. **Error handling**: Log warnings for missing/malformed files, don't crash
2. **Fallback structs**: Return partial data instead of failing completely
3. **Atomic writes**: `os.WriteFile()` is atomic for small files
4. **Explicit field extraction**: No reflection, direct struct field access
5. **Single responsibility**: Each function does one thing (extract, format, write)

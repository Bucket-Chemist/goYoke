# UX-003: Right panel: icon rail mode (< 30 cols)

## Description

When rightWidth < 30 columns, switch the right panel to an icon-only rail mode. Show status icon + 2-char abbreviation + cost for each agent. Overview collapses to 3 lines. Activity entries truncate to verb + filename.

ARCHITECTURAL DECISIONS (resolved by Staff-Architect final review):
1. **Rendering approach:** Use `Render(mode RenderMode)` unified method. `View()` delegates: `return m.Render(m.currentMode())`. This prevents dual-path drift (Risk #3) and is testable — call `Render(IconRail)` directly without constructing a full model.
2. **Scope:** Icon rail applies to `RPMAgents` mode ONLY. The 6-way `rightPanelMode` switch at `layout.go:534` means other modes (Dashboard, Settings, Telemetry, PlanPreview, Teams) keep their existing `View()` methods at all widths.

BOUNDARY HANDLING: Use hysteresis — switch at 28 (icon→text) and 32 (text→icon) to prevent flicker at the 30-col boundary.

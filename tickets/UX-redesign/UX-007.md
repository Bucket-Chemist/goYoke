# UX-007: Simple/expert toggle (hide/show right panel)

## Description

NEW RECOMMENDATION (N1 from Braintrust). Add a single keybinding (`alt+\`) to toggle the right panel on/off. When off, conversation panel gets 100% width. Preference persists to config across restarts.

This captures 80% of multi-profile PMF benefit at minimal implementation cost. Power users get the default (panels visible). Users who want a clean conversation view can opt out.

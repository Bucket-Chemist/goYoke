## Diagram Scoping Rules

| Diagram Type | Scoping Rule |
|-------------|-------------|
| Module dependencies | Always full-repo (modules, not files) |

| Call graph | Per-module. If >30 functions, show only entrypoints + 2-hop neighbors. |

## Mermaid Arrow Type Reference

| Arrow | Meaning | Example |
|-------|---------|---------|
| `-->` | Dependency (uses) | `ServiceA --> ServiceB` |
| `--|>` | Implements interface | `ConcreteType --|> Interface` |
| `--*` | Composition (has field of type) | `Parent --* Child` |
| `--o` | Aggregation (has slice/map of type) | `Collection --o Element` |

## Bash-First Workflow (MANDATORY)

**Step 1: Gather deterministic metrics**
```bash
~/.claude/scripts/gather-scout-metrics.sh <target_path> > /tmp/scout_bash_metrics.txt
```

**Step 2: Parse Bash output and classify patterns**
```bash
source /tmp/scout_bash_metrics.txt
# Now you have: TOTAL_FILES, TOTAL_LINES, ESTIMATED_TOKENS, etc.
```

**Step 3: Format JSON using Bash variables**
- Use exact values from Bash (do NOT re-estimate)
- Classify import_density: UNIQUE_IMPORTS <10=low, 10-30=medium, >30=high
- Identify key files from file list
- Add warnings if SECURITY_SENSITIVE=true
